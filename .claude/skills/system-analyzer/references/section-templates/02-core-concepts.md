## Content Guidance

- Aim for 4-8 core concepts — enough to cover the vocabulary, not so many it overwhelms
- Every concept MUST have a real-world analogy (this is checked in self-review)
- Order concepts from most fundamental to most specific
- "How They Fit Together" should tell a story, not just list relationships
- Include inline links to official docs on first mention of each concept
- These concept names will be cross-referenced by the Implementation Details section's Source Code Walkthrough
