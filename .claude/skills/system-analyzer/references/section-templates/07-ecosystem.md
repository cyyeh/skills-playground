Self-contained template for the Ecosystem & Integrations section. Level: **intermediate**.

```markdown

- [Title](URL) | github
-->

<!-- Tools built by the same team or organization.
     Include version/maturity status where relevant. -->

### Community Ecosystem
<!-- Popular third-party tools, libraries, frameworks.
     Focus on actively maintained, widely adopted projects. -->

### Common Integration Patterns
<!-- How this connects with other systems. E.g.: "Kafka + Flink for stream processing,

     Show concrete architecture snippets where helpful. -->
```

## Content Guidance

- Focus on tools and integrations that are actively maintained
- "Official Tools" should distinguish between stable/GA and experimental/beta
- "Community Ecosystem" should prioritize by adoption, not just existence
- "Common Integration Patterns" should show how this system fits into larger architectures
- Include inline links to tool/library homepages and integration guides
