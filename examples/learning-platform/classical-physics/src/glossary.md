# 詞彙表

> 古典物理學核心術語，按主題分類，附英文對照。

---

## 運動學

| 術語 | English | 定義 |
|------|---------|------|
| 位移 | Displacement | 物體位置的變化量，是有方向的向量 |
| 速度 | Velocity | 位移對時間的變化率，包含大小和方向 |
| 速率 | Speed | 速度的大小（純量），不考慮方向 |
| 加速度 | Acceleration | 速度對時間的變化率 |
| 等加速運動 | Uniform acceleration | 加速度保持不變的運動 |
| 自由落體 | Free fall | 僅受重力作用的物體運動 |
| 拋體運動 | Projectile motion | 物體以一定初速度拋出後，僅受重力的運動 |
| 參考系 | Reference frame | 用來描述物體運動的座標系統 |
| 瞬時速度 | Instantaneous velocity | 某一瞬間的速度 |
| 平均速度 | Average velocity | 總位移除以總時間 |

## 牛頓運動定律

| 術語 | English | 定義 |
|------|---------|------|
| 力 | Force | 使物體產生加速度的作用，單位為牛頓（N） |
| 慣性 | Inertia | 物體抵抗運動狀態改變的性質 |
| 質量 | Mass | 物體慣性大小的量度，單位為公斤（kg） |
| 牛頓第一定律 | Newton's first law | 不受外力時，物體保持靜止或等速直線運動 |
| 牛頓第二定律 | Newton's second law | F = ma，合力等於質量乘以加速度 |
| 牛頓第三定律 | Newton's third law | 作用力與反作用力大小相等、方向相反 |
| 合力 | Net force | 所有作用力的向量和 |
| 摩擦力 | Friction | 兩接觸面之間阻礙相對運動的力 |
| 靜摩擦力 | Static friction | 阻止物體開始滑動的摩擦力 |
| 動摩擦力 | Kinetic friction | 物體滑動時的摩擦力 |
| 正向力 | Normal force | 接觸面對物體施加的垂直支撐力 |
| 自由體圖 | Free body diagram | 顯示物體所受所有力的示意圖 |

## 功與能量

| 術語 | English | 定義 |
|------|---------|------|
| 功 | Work | 力沿位移方向的分量乘以位移，W = F·d·cosθ |
| 動能 | Kinetic energy | 物體因運動而具有的能量，KE = ½mv² |
| 位能 | Potential energy | 物體因位置或狀態而儲存的能量 |
| 重力位能 | Gravitational PE | 物體因高度而具有的位能，PE = mgh |
| 彈性位能 | Elastic PE | 彈簧因形變而儲存的能量，PE = ½kx² |
| 功能定理 | Work-energy theorem | 合力所做的功等於動能的變化量 |
| 能量守恆 | Conservation of energy | 孤立系統的總能量保持不變 |
| 功率 | Power | 做功的速率，P = W/t，單位為瓦特（W） |
| 保守力 | Conservative force | 做功只與起點終點有關、與路徑無關的力 |
| 機械能 | Mechanical energy | 動能與位能的總和 |

## 動量與衝量

| 術語 | English | 定義 |
|------|---------|------|
| 動量 | Momentum | 質量與速度的乘積，p = mv |
| 衝量 | Impulse | 力與作用時間的乘積，等於動量的變化 |
| 動量守恆 | Conservation of momentum | 不受外力時，系統總動量保持不變 |
| 彈性碰撞 | Elastic collision | 動能和動量都守恆的碰撞 |
| 非彈性碰撞 | Inelastic collision | 動量守恆但動能不守恆的碰撞 |
| 完全非彈性碰撞 | Perfectly inelastic collision | 碰撞後兩物體黏在一起的碰撞 |
| 質心 | Center of mass | 系統質量分佈的平均位置 |
| 反衝 | Recoil | 物體因釋放另一物體而向反方向運動 |

## 萬有引力

| 術語 | English | 定義 |
|------|---------|------|
| 萬有引力 | Universal gravitation | 任意兩個有質量的物體之間的吸引力 |
| 萬有引力常數 | Gravitational constant (G) | G = 6.674 × 10⁻¹¹ N·m²/kg² |
| 重力加速度 | Gravitational acceleration (g) | 地表附近約 9.8 m/s² |
| 重力場 | Gravitational field | 質量在其周圍空間產生的力場 |
| 克卜勒第一定律 | Kepler's first law | 行星軌道為橢圓，太陽在一個焦點上 |
| 克卜勒第二定律 | Kepler's second law | 行星與太陽的連線在相等時間內掃過相等面積 |
| 克卜勒第三定律 | Kepler's third law | 軌道週期的平方與半長軸的立方成正比 |
| 脫離速度 | Escape velocity | 物體脫離天體引力束縛所需的最小速度 |
| 軌道速度 | Orbital velocity | 物體在特定軌道上繞行的速度 |

## 簡諧運動與波動

| 術語 | English | 定義 |
|------|---------|------|
| 簡諧運動 | Simple harmonic motion | 回復力與位移成正比的週期性運動 |
| 振幅 | Amplitude | 振動離開平衡位置的最大距離 |
| 週期 | Period | 完成一次完整振動所需的時間 |
| 頻率 | Frequency | 單位時間內振動的次數，單位為赫茲（Hz） |
| 角頻率 | Angular frequency | ω = 2πf，單位為 rad/s |
| 彈簧常數 | Spring constant | 彈簧的勁度係數，k，單位為 N/m |
| 單擺 | Simple pendulum | 質點繫於不可伸長細線的擺動系統 |
| 波長 | Wavelength | 波的一個完整週期的空間長度 |
| 波速 | Wave speed | 波傳播的速度，v = fλ |
| 橫波 | Transverse wave | 振動方向與傳播方向垂直的波 |
| 縱波 | Longitudinal wave | 振動方向與傳播方向平行的波 |
| 疊加原理 | Superposition principle | 多個波同時經過同一點時，合成位移為各波位移之和 |
| 干涉 | Interference | 兩波疊加產生加強或減弱的現象 |
| 共振 | Resonance | 驅動頻率等於自然頻率時振幅最大的現象 |
| 駐波 | Standing wave | 兩列相同頻率的波反向傳播形成的固定波形 |

## 熱學與熱力學

| 術語 | English | 定義 |
|------|---------|------|
| 溫度 | Temperature | 物體冷熱程度的量度 |
| 熱量 | Heat | 因溫度差而傳遞的能量 |
| 比熱容 | Specific heat capacity | 單位質量物質溫度升高 1°C 所需的熱量 |
| 熱傳導 | Conduction | 透過物質直接接觸傳遞熱能 |
| 熱對流 | Convection | 透過流體流動傳遞熱能 |
| 熱輻射 | Radiation | 透過電磁波傳遞熱能 |
| 熱力學第零定律 | Zeroth law | 兩物體分別與第三物體達熱平衡，則彼此也達熱平衡 |
| 熱力學第一定律 | First law | 系統內能變化等於吸收的熱量減去對外做的功 |
| 熱力學第二定律 | Second law | 熱量不能自發地從低溫物體傳到高溫物體 |
| 熵 | Entropy | 系統混亂程度的量度 |
| 熱機 | Heat engine | 將熱能轉換為機械能的裝置 |
| 卡諾循環 | Carnot cycle | 理想熱機的最高效率循環過程 |
| 內能 | Internal energy | 系統內所有分子動能和位能的總和 |

## 流體力學

| 術語 | English | 定義 |
|------|---------|------|
| 壓力 | Pressure | 單位面積上的力，P = F/A，單位為帕斯卡（Pa） |
| 流體 | Fluid | 能夠流動的物質，包括液體和氣體 |
| 密度 | Density | 單位體積的質量，ρ = m/V |
| 帕斯卡原理 | Pascal's principle | 封閉流體中的壓力變化會等量傳遞到各處 |
| 浮力 | Buoyancy | 流體對浸入其中物體的向上推力 |
| 阿基米德原理 | Archimedes' principle | 浮力等於被排開流體的重量 |
| 白努利方程 | Bernoulli's equation | 流體沿流線的壓力、速度和高度的關係式 |
| 連續方程 | Continuity equation | 穩定流動中，截面積乘以流速為常數 |
| 層流 | Laminar flow | 流體平滑有序的流動 |
| 紊流 | Turbulent flow | 流體混亂無序的流動 |
| 表面張力 | Surface tension | 液體表面收縮的傾向 |
