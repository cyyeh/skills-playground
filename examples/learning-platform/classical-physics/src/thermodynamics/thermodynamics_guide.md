# 熱學與熱力學：入門指南

## 從一壺滾水說起

想像你在廚房裡煮水泡麵。

你打開瓦斯爐，藍色的火焰舔著鍋底。過了幾分鐘，鍋裡的水開始冒泡，蒸氣從鍋蓋的縫隙裡噴出來。你小心翼翼地打開鍋蓋——一股熱氣撲面而來，眼鏡瞬間起霧。

這個再普通不過的場景裡，藏著物理學中最深刻的一些原理：

- 火焰的熱是怎麼「跑進」水裡的？（熱傳遞）
- 為什麼水到了 100 度就開始變成蒸氣？（相變）
- 為什麼不能讓蒸氣自動變回冷水，把熱送回瓦斯桶裡？（熱力學第二定律）

要回答這些問題，你需要的就是**熱學與熱力學**。

---

## 溫度：到底多熱多冷？

先從最基本的開始——溫度。

**溫度**就是用數字來表達「有多熱」或「有多冷」。你每天看天氣預報的時候就在用它。但你有沒有想過，溫度到底在量什麼？

想像一群小朋友在教室裡。如果他們都安靜坐著不動，教室很「冷靜」。如果他們全部跑來跑去、互相碰撞，教室就很「熱鬧」。

物質內部的分子就像這些小朋友。**溫度越高，分子跑得越快、碰撞越猛烈**。溫度量的就是這種「分子有多活潑」的程度。

### 三種溫度計刻度

我們有三種主要的溫度刻度（溫標）：

- **攝氏（C）**：台灣和大多數國家日常用的。水在 0 度結冰，100 度沸騰。
- **華氏（F）**：美國人用的。水在 32 度結冰，212 度沸騰。看美國食譜時會遇到。
- **克氏（K）**：物理學家的最愛。把攝氏溫度加上 273.15 就對了。它的起點是「絕對零度」——理論上最冷最冷的溫度，分子幾乎完全不動的狀態。

記住一個簡單的轉換：K = C + 273.15。所以 0 C 就是 273.15 K，而 -273.15 C 就是 0 K——宇宙的最低溫度。

## 熱：能量在流動

現在來談「熱」。

很多人以為溫度和熱是同一回事，但它們是不同的。**溫度是狀態，熱是過程。**

打個比方：溫度像是你銀行帳戶裡的餘額，而熱量像是一筆轉帳。帳戶餘額是你「擁有」的（狀態），轉帳是金錢在「流動」（過程）。

**熱量永遠從高溫流向低溫**——就像水往低處流一樣自然。把一塊冰丟進熱咖啡裡，熱量從熱咖啡流向冰塊，咖啡變涼、冰塊融化。這不需要任何人去推動，它自己就會發生。

### 熱怎麼移動？三條路

熱有三種傳遞方式，每種你都天天遇到：

1. **傳導**——直接接觸。你摸到燙的鐵鍋，手會痛。熱從鍋子直接傳到你的手。金屬傳熱快（所以鍋子用金屬做），木頭傳熱慢（所以鍋把用木頭或塑膠做）。

2. **對流**——流動搬運。你開暖氣的時候，暖氣附近的空氣被加熱，變輕往上飄，冷空氣就從下面過來補位。整個房間的空氣這樣循環流動，慢慢就暖了。

3. **輻射**——隔空傳功。站在篝火旁邊，即使沒碰到火也覺得暖。那是因為火發出的紅外線（一種看不見的光）帶著能量穿過空氣到達你的皮膚。太陽就是這樣溫暖地球的——太陽和地球之間可是真空啊！

## 比熱：為什麼沙灘燙腳但海水不會？

去過海邊嗎？中午的沙灘燙到根本不能赤腳走，但踏進海水卻覺得涼爽。它們曬的是同一個太陽，為什麼溫度差這麼多？

答案是**比熱容**——不同物質吸收相同的熱量時，溫度升高的幅度不一樣。

水的比熱容超級大（4186 J/(kg C)），意思是它很「能忍」——吸了很多熱，溫度也不太動。砂石的比熱容小得多，吸一點熱就快速升溫。

計算熱量的公式很簡單：

**Q = mcDT**

- Q = 需要的熱量
- m = 東西有多重
- c = 比熱容（這種材料的「忍耐力」）
- DT = 溫度變化了多少

比如你要把 1 公斤的水從 20 度加熱到 100 度：Q = 1 x 4186 x 80 = 334,880 焦耳。這大概等於一個 1000 瓦的電熱水壺加熱 5 分多鐘的能量。

## 冰變水、水變氣：相變的魔法

把一塊冰從冷凍庫拿出來，放在桌上。它會慢慢融化成水。繼續加熱，水會沸騰變成蒸氣。

有趣的是，**冰在融化的過程中，溫度一直停在 0 度不動**。你給它的熱量去哪了？答案是：那些熱量全部用來打破冰晶的結構，讓分子從「排排站」變成「自由走動」。這種改變相態所需的熱叫做**潛熱**。

同樣地，水在 100 度沸騰時，溫度也不會繼續上升，所有的熱量都用來讓水分子掙脫彼此的束縛、飛入空中變成蒸氣。

這就是為什麼被 100 度的水蒸氣燙到，比被 100 度的熱水燙到更嚴重——因為蒸氣變回水時，會額外釋放大量的潛熱到你的皮膚上。

## 熱力學三大定律：宇宙的遊戲規則

### 第零定律：溫度計為什麼有用？

如果 A 和 B 都跟 C 一樣熱，那 A 和 B 也一樣熱。聽起來像廢話？但這就是為什麼溫度計能工作的原因——溫度計（C）先跟你的身體（A）達到平衡，顯示 37 度。再跟另一個人（B）達到平衡，也顯示 37 度。那你們倆的體溫就一樣。

### 第一定律：能量守恆

你給系統熱量（Q），一部分變成內能（DU），一部分做了功（W）。

**DU = Q - W**

簡單說：**能量不會無中生有，也不會憑空消失**。你給汽車加了油（Q），一部分變成車子跑的動力（W），另一部分變成引擎的熱和廢氣的熱（DU）。油箱裡那些能量一滴都不會多、也不會少。

### 第二定律：世界只會越來越亂

這是最有哲學味道的一條定律。它說：**自然界的變化，總是朝著更亂的方向走**。

物理學家用「**熵**」這個字來衡量「有多亂」。

想像你的房間。你花了一小時把它整理得乾乾淨淨。然後你什麼都不做，過了一個禮拜——它又亂了。你不需要刻意弄亂它，它自然就會亂。但要讓它變整齊，你必須花力氣。

宇宙也是這樣。**熵（亂度）只增不減。** 這就是為什麼：

- 熱咖啡會冷掉，冷咖啡不會自己變熱
- 雞蛋打碎後不會自己復原
- 你不能把一攤已經融化的冰淇淋重新凍回原來的形狀（就算重新凍，也不可能一樣）

這也是為什麼**時間有方向**——你可以分辨影片是正播還是倒播，因為倒播的世界違反了第二定律。

## 熱機：把熱變成動力

蒸汽機、汽車引擎、發電廠——這些都是**熱機**，它們的工作就是把熱能變成我們可以用的機械功。

但根據第二定律，你**不可能**把吸收的熱量百分之百變成功。一定會有一些熱量被「浪費」掉，排放到環境中。

法國工程師卡諾（Carnot）在 1824 年告訴我們，熱機效率的天花板是：

**eta = 1 - T_cold / T_hot**

T_hot 是高溫熱源的絕對溫度，T_cold 是低溫熱源的絕對溫度。

比如說，火力發電廠的蒸氣溫度是 600 K（約 327 C），冷卻水是 300 K（約 27 C）：

eta = 1 - 300/600 = 50%

這是理論上限！實際上因為摩擦、熱損失等原因，真實效率還會更低。這就是為什麼**永動機不可能存在**——不管你多聰明，都無法打敗卡諾效率這個天花板。

## 快速參考

| 概念 | 一句話總結 |
|------|-----------|
| 溫度 | 分子「有多活潑」的量度 |
| 熱量 | 因為溫差而流動的能量 |
| 比熱容 | 物質「忍受加熱」的能力，Q = mcDT |
| 潛熱 | 相變時吸收/釋放的熱量，Q = mL |
| 第零定律 | 溫度計有用的原因 |
| 第一定律 | 能量守恆：DU = Q - W |
| 第二定律 | 亂度只增不減，熱自發從高溫到低溫 |
| 熵 | 衡量亂度的物理量 |
| 卡諾效率 | 熱機效率的天花板：eta = 1 - T_cold/T_hot |
| 永動機 | 不可能。第一定律和第二定律都不允許。 |
