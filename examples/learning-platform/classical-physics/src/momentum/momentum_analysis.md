# 動量與衝量：深度解析

## 導論

動量（Momentum）是古典力學中最深刻的概念之一——它刻畫了物體「運動的慣性程度」，也是理解碰撞、爆炸、火箭推進等現象的核心工具。如果說牛頓第二定律（F = ma）是力學的心臟，那動量守恆定律就是力學的靈魂：即使在力的細節未知的情況下，我們仍能透過動量守恆推導出系統的運動結果。

從十七世紀牛頓與萊布尼茲關於「什麼才是運動的真正量度」的爭論，到今天粒子物理學家在對撞機中分析亞原子粒子的碰撞，動量的概念貫穿了整個物理學的發展。本篇將從動量的定義出發，深入探討衝量定理、動量守恆、各類碰撞、質心運動，以及反衝與火箭推進，為你建立完整的動量知識體系。

---

## 一、動量的定義

### 1.1 什麼是動量

> **動量（momentum）** 定義為物體的質量與速度的乘積，是一個向量，方向與速度相同。

**數學表達：**

**p** = m**v**

其中：
- **p** 是動量（單位：kg·m/s）
- m 是質量（單位：kg）
- **v** 是速度（單位：m/s）

**動量**的物理意義可以這樣理解：它衡量了「要讓一個運動中的物體停下來有多困難」。一輛以 30 km/h 行駛的卡車，比一輛以 30 km/h 行駛的腳踏車難停得多——因為卡車的質量遠大於腳踏車，動量也就大得多。

### 1.2 動量的基本性質

| 性質 | 說明 |
|------|------|
| 物理量類型 | **向量**（有大小和方向） |
| 單位 | kg·m/s 或 N·s |
| 方向 | 與速度方向相同 |
| 可否為零 | 可以（物體靜止時 p = 0） |
| 可否為負 | 可以（表示方向，取決於座標系的正方向定義） |
| 與動能的關係 | p = mv，KE = (1/2)mv^2 = p^2/(2m) |

### 1.3 動量與動能的區別

動量和動能都描述了物體的運動狀態，但它們是不同的物理量：

| 比較項目 | 動量 p = mv | 動能 KE = (1/2)mv^2 |
|----------|-------------|----------------------|
| 物理量類型 | **向量** | **純量** |
| 單位 | kg·m/s | J（焦耳） |
| 對速度的敏感度 | 與 v 成正比 | 與 v^2 成正比 |
| 守恆條件 | 系統不受外力時守恆 | 僅在彈性碰撞中守恆 |
| 可否為負 | 可以 | 不可以（永遠 ≥ 0） |

這個區別非常重要：在碰撞問題中，動量**總是**守恆的（前提是沒有外力），但動能**不一定**守恆。

## 二、衝量與衝量定理

### 2.1 衝量的定義

> **衝量（impulse）** 定義為力與作用時間的乘積，等於物體動量的變化量。

**數學表達：**

**J** = **F**Δt = Δ**p** = m**v**_f - m**v**_i

其中：
- **J** 是衝量（單位：N·s = kg·m/s）
- **F** 是施加的力
- Δt 是力的作用時間
- Δ**p** 是動量的變化量

### 2.2 衝量定理的推導

衝量定理可以直接從牛頓第二定律推導出來：

由 **F** = m**a** = m(Δ**v**/Δt)

兩邊同乘 Δt：

**F**Δt = mΔ**v** = Δ**p**

這就是**衝量定理**（Impulse-Momentum Theorem）：

> **作用在物體上的衝量等於物體動量的變化量。**

### 2.3 力隨時間變化的情形

在更一般的情況下，力可能隨時間變化（例如碰撞過程中，力先增大後減小）。此時衝量是力對時間的積分：

J = ∫F dt（從 t_i 到 t_f）

在力-時間圖（F-t 圖）上，**衝量等於曲線下方的面積**。這個概念在高中階段非常實用——即使不會微積分，也可以用面積法來計算衝量。

### 2.4 衝量的核心應用

衝量定理的核心啟示是：**相同的動量變化可以由大力短時間或小力長時間來實現。**

FΔt = Δp（固定）

- **大力 × 短時間**：球棒擊球，力很大但接觸時間極短
- **小力 × 長時間**：安全氣囊在碰撞中膨脹，延長了人體減速的時間，降低了人體受到的力

這就是為什麼跳遠選手落地時要屈膝——延長接觸時間，減小地面對膝蓋的衝擊力。

## 三、牛頓第二定律的動量形式

### 3.1 更一般的表述

牛頓最初在《自然哲學的數學原理》（Principia, 1687）中，就是用動量的語言來表述第二定律的：

> **物體動量的變化率等於作用在物體上的合力。**

**F** = d**p**/dt

這個形式比 F = ma 更一般、更深刻，因為它也適用於**質量變化的系統**（例如火箭——燃料不斷噴出，質量持續減少）。

在質量恆定的情況下：

**F** = d(m**v**)/dt = m(d**v**/dt) = m**a**

這就回到了我們熟悉的 F = ma。

### 3.2 為什麼動量形式更基本

| 表述形式 | F = ma | F = dp/dt |
|----------|--------|-----------|
| 適用範圍 | 質量不變的物體 | **所有情況**（包含質量變化） |
| 歷史地位 | 簡化版本 | 牛頓的原始表述 |
| 在碰撞中的應用 | 不方便（需知加速度） | **非常方便**（直接用動量變化） |
| 與守恆定律的關係 | 不直接 | 直接導出動量守恆 |

## 四、動量守恆定律

### 4.1 動量守恆的推導

考慮一個由兩個物體組成的系統，兩者之間有交互作用力，但不受外力：

由牛頓第三定律：**F**_12 = -**F**_21（物體 1 對 2 的力，等於物體 2 對 1 的力的反向）

由衝量定理：
- 物體 1 的動量變化：Δ**p**_1 = **F**_21 Δt
- 物體 2 的動量變化：Δ**p**_2 = **F**_12 Δt = -**F**_21 Δt

因此：Δ**p**_1 + Δ**p**_2 = 0

也就是說：

> **在不受外力（或合外力為零）的系統中，系統的總動量保持不變。**

**p**_1i + **p**_2i = **p**_1f + **p**_2f

m_1**v**_1i + m_2**v**_2i = m_1**v**_1f + m_2**v**_2f

### 4.2 動量守恆的適用條件

動量守恆成立的條件是**系統所受的合外力為零**（或在碰撞的極短時間內，外力的衝量可忽略）。

需要注意：
- 碰撞過程中，重力持續作用，但碰撞時間極短，重力的衝量（mgΔt）遠小於碰撞力的衝量，因此可以忽略
- 動量守恆是**向量**關係——在一維問題中，要注意正負號（方向）；在二維問題中，x 方向和 y 方向分別守恆

### 4.3 動量守恆的深層意義

從更深層的物理學角度來看，動量守恆源自**空間平移對稱性**（諾特定理，Noether's theorem）：物理定律在空間中任何位置都相同，這直接導致動量守恆。這是現代物理學最優美的結論之一。

## 五、碰撞的分類與分析

### 5.1 碰撞的分類

根據碰撞過程中動能是否守恆，碰撞可以分為三類：

| 碰撞類型 | 動量 | 動能 | 物體碰後 | 常見範例 |
|----------|------|------|----------|----------|
| **彈性碰撞** | 守恆 ✓ | 守恆 ✓ | 分開 | 撞球、氣體分子碰撞、牛頓擺 |
| **非彈性碰撞** | 守恆 ✓ | 不守恆 ✗ | 分開（但有形變或發熱） | 一般車禍、球拍擊球 |
| **完全非彈性碰撞** | 守恆 ✓ | 不守恆 ✗（損失最大） | **黏在一起** | 子彈射入木塊、兩車相撞後卡住 |

**關鍵觀念：無論哪種碰撞，動量永遠守恆（前提是合外力為零）。區別只在於動能是否守恆。**

### 5.2 彈性碰撞

在一維彈性碰撞中，兩個守恆定律同時成立：

**動量守恆：** m_1 v_1i + m_2 v_2i = m_1 v_1f + m_2 v_2f

**動能守恆：** (1/2)m_1 v_1i^2 + (1/2)m_2 v_2i^2 = (1/2)m_1 v_1f^2 + (1/2)m_2 v_2f^2

聯立求解可得碰後速度：

v_1f = [(m_1 - m_2)/(m_1 + m_2)] v_1i + [2m_2/(m_1 + m_2)] v_2i

v_2f = [2m_1/(m_1 + m_2)] v_1i + [(m_2 - m_1)/(m_1 + m_2)] v_2i

**特殊情形：**

| 情形 | 條件 | 結果 |
|------|------|------|
| 等質量碰撞 | m_1 = m_2，v_2i = 0 | v_1f = 0, v_2f = v_1i（兩者交換速度） |
| 輕撞重（靜止） | m_1 << m_2，v_2i = 0 | v_1f ≈ -v_1i（輕球反彈）, v_2f ≈ 0 |
| 重撞輕（靜止） | m_1 >> m_2，v_2i = 0 | v_1f ≈ v_1i（重球幾乎不變）, v_2f ≈ 2v_1i |

等質量彈性碰撞的「交換速度」效應，正是牛頓擺（Newton's Cradle）運作的原理。

### 5.3 完全非彈性碰撞

在完全非彈性碰撞中，兩物體碰後黏在一起，以相同速度運動：

**動量守恆：** m_1 v_1i + m_2 v_2i = (m_1 + m_2) v_f

碰後速度：v_f = (m_1 v_1i + m_2 v_2i) / (m_1 + m_2)

**動能損失：**

ΔKE = KE_i - KE_f = (1/2) × [m_1 m_2 / (m_1 + m_2)] × (v_1i - v_2i)^2

動能損失轉化為熱能、聲能和形變能。完全非彈性碰撞中，動能的損失是所有碰撞類型中最大的。

### 5.4 非彈性碰撞

一般的非彈性碰撞介於彈性碰撞和完全非彈性碰撞之間。我們可以用**恢復係數**（coefficient of restitution, e）來描述碰撞的「彈性程度」：

e = |v_2f - v_1f| / |v_1i - v_2i| = 碰後分離速率 / 碰前靠近速率

| 恢復係數 e 的值 | 碰撞類型 |
|-----------------|----------|
| e = 1 | 完全彈性碰撞 |
| 0 < e < 1 | 非彈性碰撞 |
| e = 0 | 完全非彈性碰撞 |

## 六、二維碰撞

### 6.1 二維碰撞的基本原理

在二維碰撞中，動量守恆是**向量方程**，需要在 x 方向和 y 方向分別列式：

**x 方向：** m_1 v_1ix + m_2 v_2ix = m_1 v_1fx + m_2 v_2fx

**y 方向：** m_1 v_1iy + m_2 v_2iy = m_1 v_1fy + m_2 v_2fy

### 6.2 經典範例：撞球

撞球是二維碰撞的經典例子。當母球以速度 v_0 斜向撞擊靜止的目標球時：

在等質量彈性碰撞中，碰後兩球的速度方向夾角為 90 度。這個結論可以用動量守恆和動能守恆的向量關係推導出來：

由動量守恆（向量）：m**v**_0 = m**v**_1f + m**v**_2f → **v**_0 = **v**_1f + **v**_2f

由動能守恆：(1/2)mv_0^2 = (1/2)mv_1f^2 + (1/2)mv_2f^2 → v_0^2 = v_1f^2 + v_2f^2

第二個方程是畢氏定理的形式，這意味著 **v**_1f 和 **v**_2f 互相垂直，夾角為 90 度。

## 七、質心

### 7.1 質心的定義

> **質心（center of mass, CM）** 是系統質量分佈的加權平均位置，系統在外力作用下，質心的運動遵循牛頓第二定律。

**二粒子系統的質心：**

x_CM = (m_1 x_1 + m_2 x_2) / (m_1 + m_2)

### 7.2 質心的運動

質心的速度：

**v**_CM = (m_1 **v**_1 + m_2 **v**_2) / (m_1 + m_2) = **p**_total / M_total

這裡有一個極其重要的結論：

> **系統的總動量等於「全部質量集中在質心」時的動量。**

如果系統不受外力，質心的速度不變（做等速直線運動或靜止）。即使系統內部發生碰撞、爆炸，質心的運動都不受影響。

### 7.3 質心參考系

在質心參考系中觀察碰撞，問題會大幅簡化：
- 碰撞前後，系統的總動量為零
- 彈性碰撞中，兩物體的速度大小不變、方向反轉
- 這個概念在粒子物理中尤其重要（例如 CERN 的對撞機就是讓粒子在質心參考系中碰撞，以最大化碰撞能量）

## 八、反衝與火箭推進

### 8.1 反衝效應

> **反衝（recoil）** 是動量守恆在「系統內部爆發」情境中的應用：初始動量為零的系統，一部分向一個方向運動時，另一部分必然向反方向運動。

常見例子：
- 開槍時，子彈向前飛出，槍身向後退（後座力）
- 溜冰者推牆後向後滑動
- 太空人在太空中丟出工具，自己會向反方向飄移

**數學表達：** 0 = m_1 **v**_1 + m_2 **v**_2 → **v**_1 = -(m_2/m_1) **v**_2

### 8.2 火箭推進原理

火箭是反衝效應最壯觀的應用。火箭向後噴出高速燃氣，由動量守恆，火箭本體就獲得向前的動量。

**齊奧爾科夫斯基火箭方程式（Tsiolkovsky rocket equation）：**

Δv = v_e × ln(m_i / m_f)

其中：
- Δv 是火箭獲得的速度變化量
- v_e 是噴出氣體相對於火箭的速度（排氣速度）
- m_i 是火箭的初始質量（含燃料）
- m_f 是火箭燃燒完畢後的最終質量
- ln 是自然對數

這個方程告訴我們：
- **排氣速度越高**，火箭獲得的速度越大（因此要用高能燃料）
- **質量比 m_i/m_f 越大**，火箭獲得的速度越大（因此火箭的大部分質量都是燃料）
- 速度的增加與質量比的**對數**成正比——這意味著每多載一點燃料的效益是遞減的，這就是為什麼大型火箭通常採用**多級火箭**設計（用完的空殼直接丟棄，減少質量）

### 8.3 火箭推進的現實挑戰

| 挑戰 | 說明 |
|------|------|
| 質量比限制 | 現有化學燃料的排氣速度有上限（約 2-4.5 km/s），要達到太空所需的速度（約 8 km/s 入軌），需要極大的質量比 |
| 多級火箭 | 將火箭分成多級，每級用完後拋棄空殼，減少需要加速的質量 |
| 引力損失 | 火箭在上升過程中要對抗地球引力，實際所需 Δv 遠大於軌道速度 |
| 離子推進 | 使用電場加速離子，排氣速度可達 30-50 km/s（遠大於化學火箭），但推力很小，需要長時間加速 |

## 九、動量在現代物理中的地位

### 9.1 從古典到量子

動量的概念遠不止於古典力學。在現代物理中：

- **量子力學：** 動量是最基本的物理量之一。海森堡不確定原理指出，粒子的位置和動量不能同時被精確測量：Δx·Δp ≥ ℏ/2
- **狹義相對論：** 在接近光速時，動量的公式修正為 p = γmv，其中 γ = 1/√(1 - v^2/c^2) 是勞倫茲因子
- **光子的動量：** 光子沒有靜質量，但有動量 p = h/λ = E/c（這就是太陽帆和光壓的物理基礎）

### 9.2 守恆定律的統一圖景

| 守恆量 | 對應的對稱性 | 適用條件 |
|--------|-------------|----------|
| 動量 | 空間平移對稱性 | 合外力為零 |
| 能量 | 時間平移對稱性 | 孤立系統 |
| 角動量 | 旋轉對稱性 | 合外力矩為零 |

這三大守恆定律是整個物理學的基石，而動量守恆是其中最常使用、最直接實用的一個。
