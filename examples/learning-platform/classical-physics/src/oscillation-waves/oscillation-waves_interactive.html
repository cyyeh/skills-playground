原理說明：根據疊加原理，兩列波在同一點的合位移等於各波位移的代數和。當兩波同相（相位差 = 0 度），波峰對波峰，產生建設性干涉，合振幅最大。當兩波反相（相位差 = 180 度），波峰對波谷，產生破壞性干涉，合振幅最小。
將兩波設為相同頻率和振幅，然後慢慢調整相位差從 0 度到 180 度，觀察疊加波的振幅如何從最大變到最小。也可以設定微小的頻率差（例如 1.0 Hz 和 1.1 Hz），觀察「拍頻」現象——合成波的振幅會週期性地變大變小。

原理說明：彈簧系統的週期 T = 2π√(m/k)。質量越大、彈簧越軟（k 越小），週期越長。注意觀察：位移最大時速度為零（動能 = 0），通過平衡位置時速度最大（位能 = 0），而總能量始終不變。
先固定彈簧常數，只改變質量，觀察週期如何變化。然後固定質量，改變彈簧常數。你會發現振幅不影響週期——這就是「等時性」。

原理說明：單擺週期公式 T = 2π√(L/g) 只在小角度（通常 < 15 度）下精確。當初始角度增大時，實際週期會比公式預測的稍微長一些。試著增大角度，觀察「實測週期」和「公式預測」的差距如何變化。
先設定小角度（10 度），看看實測週期和公式預測幾乎一致。然後把角度加大到 60 度甚至 80 度，觀察誤差變大。這就是為什麼我們說公式只在「小角度近似」下成立。另外試試改變擺長——擺越長，週期越長。

原理說明：兩端固定的弦只能產生特定頻率的駐波。第 n 諧波的弦上有 n+1 個節點和 n 個腹點。頻率 fₙ = n × f₁，其中 f₁ 是基頻。觀察隨著諧波數增加，節點和腹點的數量如何變化。
從基頻（第 1 諧波）開始，慢慢增加頻率倍數。注意觀察：第 n 諧波恰好有 n 個「肚子」（腹點）。弦的兩端永遠是節點（不動）。第 2 諧波的正中間多了一個節點，把弦分成兩段。
