# 簡諧運動與波動：深度解析

## 導論

簡諧運動（Simple Harmonic Motion, SHM）與波動（Waves）是古典物理學中最優美的主題之一。從鐘擺的規律擺盪，到吉他弦的振動發聲，再到光波穿越宇宙抵達我們的眼睛——振盪與波動的現象無處不在。

這個主題之所以重要，是因為它連結了力學、聲學、光學，甚至電磁學的核心概念。牛頓運動定律告訴我們物體如何因力而運動；而簡諧運動則展示了當一個物體反覆受到**回復力**的作用時，會產生怎樣週期性的美妙運動。當這種振盪在介質中傳遞時，就形成了波。

本篇將從簡諧運動的定義出發，深入探討彈簧系統、單擺、能量轉換，再延伸到波的基本性質、疊加原理、干涉、駐波，以及都卜勒效應，為你建立完整而紮實的知識體系。

---

## 一、簡諧運動的定義與特徵

### 1.1 什麼是簡諧運動？

> **簡諧運動**（Simple Harmonic Motion, SHM）是一種特殊的週期性運動，其中物體所受的**回復力**與位移成正比，且方向指向平衡位置。

用數學語言表達，簡諧運動的條件是：

**F = -kx**

其中 F 是回復力，k 是比例常數（對彈簧系統而言就是彈簧常數），x 是偏離平衡位置的位移，負號表示力的方向永遠指向平衡位置。

### 1.2 簡諧運動的核心特徵

| 特徵 | 說明 |
|------|------|
| **週期性** | 運動反覆重複，有固定的週期 |
| **回復力正比於位移** | 偏離越遠，回復力越大 |
| **平衡位置** | 物體在此處淨力為零，速度最大 |
| **對稱性** | 物體在平衡位置兩側的運動對稱 |
| **振幅恆定**（理想情況） | 在無阻尼的理想情況下，振幅不隨時間改變 |

## 二、彈簧振動與虎克定律

### 2.1 虎克定律

**虎克定律**（Hooke's Law）是描述彈簧行為的基本定律：

**F = -kx**

- **F**：彈簧對物體施加的回復力（單位：牛頓 N）
- **k**：彈簧常數（單位：N/m），代表彈簧的「硬度」
- **x**：偏離自然長度的位移（單位：公尺 m）
- **負號**：力的方向永遠與位移方向相反

> 虎克定律有其適用範圍——只有在彈簧未超過**彈性極限**的前提下才成立。超過彈性極限，彈簧會發生永久形變，不再遵循線性關係。

### 2.2 彈簧振盪系統

當一個質量為 m 的物體接在彈簧上，被拉離平衡位置後釋放，物體就會做簡諧運動。根據牛頓第二定律：

F = ma = -kx

因此加速度 a = -(k/m)x

這告訴我們：加速度與位移成正比，方向相反——這正是簡諧運動的判據。

### 2.3 彈簧系統的週期

彈簧振盪系統的**週期**為：

**T = 2π√(m/k)**

- **T**：振盪一個完整週期所需的時間（單位：秒 s）
- **m**：物體質量（單位：公斤 kg）
- **k**：彈簧常數（單位：N/m）

> **重要觀念：** 彈簧振盪的週期只取決於質量和彈簧常數，**與振幅無關**。這叫做**等時性**——不管你把彈簧拉開多遠或多近，它來回一趟的時間是一樣的（前提是振幅不超過彈性極限）。

## 三、簡諧運動的數學描述

### 3.1 位移方程

簡諧運動的位移可以用正弦或餘弦函數精確描述：

**x(t) = A sin(ωt + φ)**

| 符號 | 名稱 | 意義 | 單位 |
|------|------|------|------|
| x(t) | 位移 | 物體在時間 t 的偏離位置 | m |
| A | **振幅** | 最大偏離平衡位置的距離 | m |
| ω | **角頻率** | 每秒掃過的弧度數 | rad/s |
| t | 時間 | 從計時起點開始的時間 | s |
| φ | **初相位** | 描述 t=0 時物體的初始狀態 | rad |

### 3.2 速度與加速度

從位移方程可以推導出速度與加速度：

- **速度：** v(t) = dx/dt = Aω cos(ωt + φ)
- **加速度：** a(t) = dv/dt = -Aω² sin(ωt + φ) = -ω²x

速度的最大值為 **v_max = Aω**，出現在平衡位置處（x = 0）。

加速度的最大值為 **a_max = Aω²**，出現在最大位移處（x = ±A）。

### 3.3 位移、速度、加速度的關係

> 在簡諧運動中，**速度**領先**位移** π/2（90度），而**加速度**與**位移**反相（相差 π 即 180 度）。

這意味著：
- 當位移最大時（物體在端點），速度為零，加速度最大
- 當位移為零時（物體經過平衡位置），速度最大，加速度為零

## 四、週期、頻率與角頻率的關係

### 4.1 三者的定義與換算

| 物理量 | 符號 | 定義 | 單位 |
|--------|------|------|------|
| **週期** | T | 完成一次完整振動所需的時間 | 秒（s） |
| **頻率** | f | 每秒振動的次數 | 赫茲（Hz） |
| **角頻率** | ω | 每秒掃過的弧度 | rad/s |

三者之間的關係：

- **f = 1/T**
- **ω = 2πf = 2π/T**

### 4.2 實例

一個彈簧系統每 0.5 秒完成一次振動：
- 週期 T = 0.5 s
- 頻率 f = 1/0.5 = 2 Hz（每秒振動 2 次）
- 角頻率 ω = 2π × 2 = 4π ≈ 12.57 rad/s

## 五、單擺的運動

### 5.1 簡單擺的結構

**單擺**（Simple Pendulum）由一條不可伸縮、質量可忽略的繩子，懸掛一個小而重的擺錘所組成。當擺錘被拉離鉛直平衡位置後釋放，在重力的作用下會來回擺動。

### 5.2 小角度近似

嚴格來說，單擺的運動並不是完美的簡諧運動。但當擺角 θ 很小時（通常 θ < 15°），我們可以使用小角度近似：

sin θ ≈ θ（θ 以弧度表示）

在這個近似下，單擺的回復力近似正比於位移，因此可以視為簡諧運動。

### 5.3 單擺的週期公式

**T = 2π√(L/g)**

- **T**：週期（秒）
- **L**：擺長——從懸掛點到擺錘質心的距離（公尺）
- **g**：重力加速度（約 9.8 m/s²）

> **關鍵觀念：** 單擺的週期只與**擺長**和**重力加速度**有關，與**擺錘質量無關**，也與**振幅無關**（小角度下）。這就是伽利略最早觀察到的擺的等時性。

| 比較 | 彈簧振盪 T = 2π√(m/k) | 單擺 T = 2π√(L/g) |
|------|------------------------|---------------------|
| 影響週期的因素 | 質量 m、彈簧常數 k | 擺長 L、重力加速度 g |
| 與質量的關係 | 質量越大，週期越長 | 與質量無關 |
| 與振幅的關係 | 無關 | 無關（小角度） |

## 六、能量在簡諧運動中的轉換

### 6.1 動能與位能的互換

在簡諧運動中，能量不斷在**動能**（Kinetic Energy, KE）和**位能**（Potential Energy, PE）之間轉換：

- **彈性位能：** PE = (1/2)kx²
- **動能：** KE = (1/2)mv²
- **總機械能：** E = KE + PE = (1/2)kA²（恆定）

### 6.2 能量轉換的過程

| 物體位置 | 位移 x | 速度 v | 動能 KE | 位能 PE |
|----------|--------|--------|---------|---------|
| 最大位移 x = A | 最大 | 零 | **零** | **最大** |
| 平衡位置 x = 0 | 零 | 最大 | **最大** | **零** |
| 最大位移 x = -A | 最大 | 零 | **零** | **最大** |

> **核心概念：** 在理想（無摩擦、無空氣阻力）的情況下，簡諧運動的**總機械能守恆**。能量只在動能和位能之間來回轉換，總和永遠不變。

## 七、波的基本性質

### 7.1 什麼是波？

> **波**（Wave）是一種能量和擾動在介質中傳遞的現象，但**介質本身並不隨波移動**。

當你在池塘裡丟一顆石頭，水面上會出現向外擴散的波紋。水面的每一個點只是在上下振動，水並沒有真的從石頭落點流向池塘邊緣——傳遞出去的是**能量**和**振動的模式**。

### 7.2 波的基本物理量

| 物理量 | 符號 | 定義 | 單位 |
|--------|------|------|------|
| **波長** | λ（lambda） | 相鄰兩個相同相位點之間的距離 | 公尺（m） |
| **頻率** | f | 介質中某一點每秒振動的次數 | 赫茲（Hz） |
| **週期** | T | 介質中某一點完成一次完整振動的時間 | 秒（s） |
| **振幅** | A | 振動偏離平衡位置的最大距離 | 公尺（m） |
| **波速** | v | 波在介質中傳播的速度 | m/s |

### 7.3 波速公式

**v = fλ = λ/T**

這是波動中最基本也最重要的關係式。波速由介質的性質決定，而頻率由振動源決定。

## 八、橫波與縱波

### 8.1 兩種基本波形

| 比較項目 | **橫波**（Transverse Wave） | **縱波**（Longitudinal Wave） |
|----------|----------------------------|-------------------------------|
| 振動方向 | **垂直**於波的傳播方向 | **平行**於波的傳播方向 |
| 典型例子 | 繩波、水面波、電磁波 | 聲波、彈簧縱波（疏密波） |
| 特徵結構 | 波峰（crest）與波谷（trough） | 壓縮（compression）與稀疏（rarefaction） |
| 在固體中 | 可以傳播 | 可以傳播 |
| 在液體/氣體中 | 一般不能傳播（電磁波例外） | 可以傳播 |

### 8.2 聲波是縱波

聲音的傳播是空氣分子的疏密振動。當音叉振動時，它交替壓縮和拉開前方的空氣，形成一系列「密部」和「疏部」向外傳播。每個空氣分子只在其平衡位置附近前後振動，而不是真的隨著聲音跑到你的耳朵裡。

### 8.3 水面波的特殊性

水面波比較特殊——水分子的運動其實是**圓周運動**，同時包含橫向和縱向的分量。所以嚴格來說，水面波既不是純粹的橫波，也不是純粹的縱波。

## 九、波的疊加原理

### 9.1 疊加原理的定義

> **疊加原理**（Principle of Superposition）：當兩個或多個波同時通過介質中的同一點時，該點的合位移等於各個波在該點產生的位移之**代數和**。

**y_合 = y₁ + y₂ + y₃ + ...**

這個原理之所以成立，是因為波動方程是**線性**的。疊加完成後，各波繼續按原來的方式傳播，互不影響。

### 9.2 疊加的日常例子

想像你和朋友在池塘的兩側各丟一顆石頭。兩組波紋會交會、重疊、穿過彼此，然後繼續各自傳播——好像對方的波根本不存在一樣。在交會的瞬間，水面的高度就是兩組波各自高度的總和。

## 十、建設性干涉與破壞性干涉

### 10.1 干涉現象

**干涉**（Interference）是波的疊加原理的直接結果。當兩列頻率相同的波相遇時：

| 干涉類型 | 條件 | 結果 | 路程差 |
|----------|------|------|--------|
| **建設性干涉** | 兩波同相（波峰對波峰） | 振幅**增大**（最大為 A₁ + A₂） | nλ（n = 0, 1, 2, ...） |
| **破壞性干涉** | 兩波反相（波峰對波谷） | 振幅**減小**（最小為 |A₁ - A₂|） | (n + 1/2)λ |

### 10.2 生活中的干涉

- **建設性干涉的例子：** 在音樂廳裡，某些位置聽起來音量特別大，是因為從不同方向來的聲波恰好同相疊加
- **破壞性干涉的例子：** 抗噪耳機（Active Noise Cancellation）的原理就是產生一個與噪音相位相反的聲波，兩者疊加後互相抵消

## 十一、駐波與共振

### 11.1 駐波的形成

> **駐波**（Standing Wave）是兩列振幅、頻率、波長相同但傳播方向相反的波疊加而成的特殊波形。

駐波不像行進波那樣在空間中移動，而是「站在原地」振動。

### 11.2 節點與腹點

| 特殊位置 | 名稱 | 特徵 |
|----------|------|------|
| 振幅永遠為零的點 | **節點**（node） | 始終靜止不動 |
| 振幅最大的點 | **腹點**（antinode） | 以最大振幅振動 |

相鄰兩個節點之間的距離為 **λ/2**。

### 11.3 弦上的駐波與諧波

在兩端固定的弦（例如吉他弦）上，只有特定頻率的波才能形成駐波。這些特定頻率稱為**諧波**（harmonics）或**泛音**：

| 諧波 | 名稱 | 弦長與波長的關係 | 頻率 |
|------|------|------------------|------|
| 第一諧波 | **基頻** | L = λ₁/2 → λ₁ = 2L | f₁ = v/(2L) |
| 第二諧波 | 第一泛音 | L = λ₂ → λ₂ = L | f₂ = 2f₁ |
| 第三諧波 | 第二泛音 | L = 3λ₃/2 → λ₃ = 2L/3 | f₃ = 3f₁ |
| 第 n 諧波 | — | L = nλₙ/2 → λₙ = 2L/n | fₙ = nf₁ |

### 11.4 共振

> **共振**（Resonance）發生在外力的頻率恰好等於系統的**自然頻率**（或其整數倍）時。此時系統會以最大振幅振動，能量傳遞效率最高。

共振在生活中無處不在：
- 推鞦韆時，如果推的節奏和鞦韆擺動的頻率一致，鞦韆會越擺越高
- 歌手唱出特定音高可以震碎酒杯——因為聲波的頻率恰好等於酒杯的自然振動頻率
- 樂器利用共鳴箱來放大聲音

但共振也可能是危險的——如果建築物或橋梁被迫以其自然頻率振動，可能導致結構破壞。

## 十二、都卜勒效應簡介

### 12.1 什麼是都卜勒效應？

> **都卜勒效應**（Doppler Effect）是指波源與觀察者之間有相對運動時，觀察者接收到的頻率與波源發出的頻率不同的現象。

### 12.2 都卜勒效應的規律

| 情境 | 觀察到的頻率變化 | 日常經驗 |
|------|------------------|----------|
| 波源**靠近**觀察者 | 頻率**升高**（波長變短） | 救護車靠近時，鳴笛聲音尖銳 |
| 波源**遠離**觀察者 | 頻率**降低**（波長變長） | 救護車遠離時，鳴笛聲音低沉 |

### 12.3 都卜勒效應公式

當波源和觀察者沿同一直線運動時：

**f' = f × (v ± v_觀察者) / (v ∓ v_波源)**

其中：
- f' 是觀察者接收到的頻率
- f 是波源發出的頻率
- v 是波在介質中的速度
- v_觀察者 是觀察者的速度
- v_波源 是波源的速度

上式中的正負號規則：觀察者朝波源移動取「+」，遠離取「-」；波源朝觀察者移動取「-」，遠離取「+」。

### 12.4 都卜勒效應的應用

- **天文學：** 透過星光的**紅移**（頻率降低）或**藍移**（頻率升高），天文學家可以判斷恆星和星系是在遠離還是靠近地球。哈伯正是利用紅移現象發現了宇宙正在膨脹。
- **醫學：** 都卜勒超音波用於測量血液流速和胎兒心跳。
- **交通：** 測速雷達利用都卜勒效應測量車輛速度。
- **氣象學：** 都卜勒雷達用於追蹤風暴和降雨的移動。

## 總結

本章涵蓋了從簡諧運動到波動的完整知識鏈：

1. **簡諧運動**是回復力正比於位移的週期性運動，數學上用正弦函數描述
2. **彈簧系統**和**單擺**是兩個最經典的簡諧運動模型
3. **能量守恆**在簡諧運動中體現為動能與位能的持續互換
4. **波**是振盪在介質中的傳遞，分為橫波和縱波
5. **疊加原理**是理解干涉和駐波的基礎
6. **建設性干涉**使振幅增大，**破壞性干涉**使振幅減小
7. **駐波**是兩列反向行進波的疊加，**共振**發生在外力頻率等於自然頻率時
8. **都卜勒效應**描述了波源與觀察者相對運動時的頻率變化

掌握這些概念，你將擁有理解聲學、光學、甚至現代量子力學的堅實基礎。振盪與波動的思維方式，將在你的物理學之旅中反覆出現。
