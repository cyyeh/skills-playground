# 運動學：深度解析

## 導論

運動學（Kinematics）是古典力學的第一塊基石——它專注於描述物體「如何運動」，而暫時不追問「為什麼運動」。在還沒有引入力的概念之前，我們先學會用精確的數學語言刻畫位置、速度與加速度的變化，這是理解整個力學體系的必要前提。

從伽利略在比薩斜塔旁進行的傳奇實驗，到今天工程師計算火箭軌道，運動學的核心觀念貫穿了四百年的物理學發展史。本篇將從最基本的位移與距離出發，逐步深入等加速運動、自由落體、拋體運動與相對運動，為後續學習牛頓運動定律打下堅實的基礎。

---

## 一、位置、位移與距離

### 1.1 參考系與座標

要描述運動，首先需要一個**參考系**（reference frame）——也就是選定一個觀察者的立場和一套座標系統。例如，我們通常以地面為參考系，建立一維的 x 軸或二維的 xy 平面。

**位置**（position）是物體在某一時刻相對於原點的座標值，通常以符號 x（一維）或 (x, y)（二維）表示。

### 1.2 位移與距離的區別

> **位移（displacement）** 是從起點指向終點的向量，只關心起點與終點的相對位置，與路徑無關。  
> **距離（distance）** 是物體實際行經的路徑長度，是純量（只有大小，沒有方向）。

這兩個概念是初學者最容易混淆的地方。以下表格清楚整理它們的差異：

| 比較項目 | 位移 | 距離 |
|----------|------|------|
| 符號 | Δx 或 **d** (向量) | d (純量) |
| 物理量類型 | **向量**（有大小和方向） | **純量**（只有大小） |
| 計算方式 | 終點位置 - 起點位置 | 路徑長度的總和 |
| 可否為負 | 可以（表示方向） | 不可以（永遠 ≥ 0） |
| 可否為零 | 可以（回到原點） | 只在完全沒移動時為零 |
| 範例 | 你從家走到學校再走回家，位移 = 0 | 你從家走到學校再走回家，距離 = 來回兩趟的路程 |

**數學表達：**

一維位移：Δx = x_f - x_i

其中 x_f 是終點位置，x_i 是起點位置。

## 二、速度與速率

### 2.1 平均速度與平均速率

> **平均速度（average velocity）** = 位移 / 時間，是向量。  
> **平均速率（average speed）** = 距離 / 時間，是純量。

**平均速度：** v_avg = Δx / Δt = (x_f - x_i) / (t_f - t_i)

**平均速率：** 平均速率 = 總距離 / 總時間

| 比較項目 | 速度（velocity） | 速率（speed） |
|----------|-------------------|---------------|
| 物理量類型 | **向量** | **純量** |
| 定義基礎 | 位移 / 時間 | 距離 / 時間 |
| 可否為負 | 可以 | 不可以 |
| 可否為零 | 可以（回到原點時平均速度為零） | 只在靜止時為零 |
| 常用單位 | m/s | m/s |

### 2.2 瞬時速度

**瞬時速度**（instantaneous velocity）是在某一極短時刻的速度，數學上是位移對時間的微分：

v = lim(Δt→0) Δx/Δt = dx/dt

在高中階段，我們通常不直接使用微積分，但要理解：「瞬時速度就是當時間間隔趨近於零時的平均速度」。在 x-t 圖（位置—時間圖）上，瞬時速度就是該點切線的斜率。

### 2.3 速度的圖形意義

- **x-t 圖的斜率** = 速度
- **v-t 圖的面積** = 位移（面積在 x 軸下方則為負位移）
- **v-t 圖的斜率** = 加速度

這三句話是運動學圖形分析的核心，務必牢記。

## 三、加速度

### 3.1 定義

> **加速度（acceleration）** 是速度對時間的變化率，描述速度改變的快慢。

**平均加速度：** a_avg = Δv / Δt = (v_f - v_i) / (t_f - t_i)

**瞬時加速度：** a = dv/dt

加速度的單位是 m/s^2（公尺每秒平方）。

### 3.2 加速度的方向與意義

常見的迷思是「加速度就是加速」。事實上：

- 加速度**與速度同方向** → 物體加速（速率增大）
- 加速度**與速度反方向** → 物體減速（速率減小）
- 加速度方向改變 → 速度方向改變（例如圓周運動中的向心加速度）

**重要觀念：** 加速度為零不代表物體靜止，而是代表物體做等速運動（速度不變）。

## 四、等加速運動公式

### 4.1 五大公式

當加速度 a 為常數時，我們有以下五個基本公式（這是高中物理最核心的公式組之一）：

| 編號 | 公式 | 不含的變數 |
|------|------|------------|
| 1 | v = v_0 + at | d（位移） |
| 2 | d = v_0 t + (1/2)at^2 | v（末速） |
| 3 | v^2 = v_0^2 + 2ad | t（時間） |
| 4 | d = (v_0 + v)/2 × t | a（加速度） |
| 5 | d = vt - (1/2)at^2 | v_0（初速） |

其中：
- v_0 = 初速度
- v = 末速度
- a = 加速度
- t = 時間
- d = 位移

### 4.2 如何選擇公式

解題的關鍵在於辨認**已知量**和**求解量**，然後選擇「包含所有已知量和求解量」的公式。實用技巧是看表格中「不含的變數」——如果某個變數是你既不知道也不需要的，就選那個公式。

**範例：** 一輛汽車以 10 m/s 的初速度加速，加速度為 2 m/s^2，求經過 5 秒後的位移。

已知：v_0 = 10 m/s, a = 2 m/s^2, t = 5 s  
求：d  
不需要的變數：v（末速）  
選擇公式 2：d = v_0 t + (1/2)at^2 = 10(5) + (1/2)(2)(5^2) = 50 + 25 = 75 m

### 4.3 v-t 圖與等加速運動

等加速運動在 v-t 圖上呈現為一條**直線**（斜率 = 加速度）。位移等於圖形下方的面積（梯形面積），可驗證公式 4：

d = (v_0 + v)/2 × t = 梯形面積

## 五、自由落體

### 5.1 定義與條件

> **自由落體（free fall）** 是物體僅受重力作用、不受空氣阻力影響的運動。

自由落體是等加速運動的一個特例，加速度恆為重力加速度 g：

**g ≈ 9.8 m/s^2**（地表附近，方向向下）

在計算中，我們通常取 g = 9.8 m/s^2 或 g ≈ 10 m/s^2（簡化計算）。

### 5.2 自由落體公式

將等加速運動公式中的 a 換成 g（取向下為正），可得：

- v = v_0 + gt
- h = v_0 t + (1/2)gt^2（h 為向下的位移）
- v^2 = v_0^2 + 2gh

若物體**從靜止開始下落**（v_0 = 0）：

- v = gt
- h = (1/2)gt^2
- v^2 = 2gh

### 5.3 重要結論

1. **所有物體在真空中的自由落體加速度相同**，與物體的質量、大小、形狀無關。這是伽利略的偉大發現。
2. 自由落體 1 秒後速度約 9.8 m/s，下落距離約 4.9 m。
3. 自由落體的速度隨時間**線性增加**，距離隨時間的**平方**增加。

| 下落時間 (s) | 速度 (m/s) | 累計下落距離 (m) |
|:---:|:---:|:---:|
| 0 | 0 | 0 |
| 1 | 9.8 | 4.9 |
| 2 | 19.6 | 19.6 |
| 3 | 29.4 | 44.1 |
| 4 | 39.2 | 78.4 |
| 5 | 49.0 | 122.5 |

## 六、拋體運動

### 6.1 基本原理

> **拋體運動（projectile motion）** 是物體在重力場中，僅受重力作用下的二維運動。

關鍵觀念：拋體運動可以分解為**水平方向**和**鉛直方向**兩個獨立的運動：

- **水平方向：** 等速運動（無水平加速度，忽略空氣阻力）
- **鉛直方向：** 等加速運動（加速度 = g，向下）

這兩個方向的運動**彼此獨立**，互不影響。

### 6.2 水平拋射

物體以初速度 v_0 水平射出（例如從桌邊推落一顆球）：

**水平方向：**
- x = v_0 t

**鉛直方向（向下為正）：**
- v_y = gt
- y = (1/2)gt^2

**合速度：** v = sqrt(v_0^2 + v_y^2)

**軌跡方程：** y = g/(2v_0^2) × x^2（拋物線）

**落地時間** 只取決於下落高度 h：t = sqrt(2h/g)

**水平射程：** R = v_0 × t = v_0 × sqrt(2h/g)

### 6.3 斜拋運動

物體以初速度 v_0、仰角 θ 射出：

**初速度分解：**
- v_0x = v_0 cos θ（水平分量）
- v_0y = v_0 sin θ（鉛直分量）

**水平方向：**
- x = v_0 cos θ × t

**鉛直方向（向上為正）：**
- v_y = v_0 sin θ - gt
- y = v_0 sin θ × t - (1/2)gt^2

**重要結論：**

| 物理量 | 公式 |
|--------|------|
| 最高點高度 | H = (v_0 sin θ)^2 / (2g) |
| 飛行總時間 | T = 2v_0 sin θ / g |
| 水平射程 | R = v_0^2 sin 2θ / g |
| 最大射程角度 | θ = 45° 時，R 最大 |

**斜拋的對稱性：** 在不考慮空氣阻力的情況下，斜拋軌跡是對稱的拋物線——上升和下降的時間相等，最高點恰在水平射程的正中央。

**互補角射程相同：** 拋射角 θ 和 (90° - θ) 的射程相同。例如，30° 和 60° 拋出的物體，水平射程一樣（但軌跡形狀不同）。

## 七、相對運動

### 7.1 基本概念

> **相對運動**是從不同參考系觀察同一物體運動時，得到不同速度的現象。

在日常生活中，我們無時無刻不在經歷相對運動。坐在行駛的火車上，你覺得自己靜止不動，但月台上的人看你正在高速移動。

### 7.2 相對速度公式

設 A、B 為兩個運動的物體，則：

**A 相對於 B 的速度 = A 的速度 - B 的速度**

v_AB = v_A - v_B

**一維的情況：**
- 兩人同向行走：相對速度 = 兩者速度之差（較快的追較慢的）
- 兩人對向行走：相對速度 = 兩者速度之和（互相接近的速度）

**二維的情況：**

在二維平面上，相對速度需要用向量相減：

v_AB = **v_A** - **v_B**

需要利用向量分解或向量三角形來求解。

### 7.3 範例

一艘船在靜水中的速度為 5 m/s（向東），河水流速為 3 m/s（向南）。對岸上的人來說，船的合速度為：

v = sqrt(5^2 + 3^2) = sqrt(34) ≈ 5.83 m/s

方向：tan θ = 3/5，θ ≈ 31° 偏南。

因此，船實際上不是正東前進，而是偏向東南方。這就是為什麼渡河問題中，船必須將船頭偏向上游，才能到達正對岸的目標。

## 總結

運動學提供了一套完整的數學語言來描述物體的運動：

| 核心概念 | 關鍵公式或原理 |
|----------|----------------|
| 位移 vs 距離 | 位移是向量，距離是純量 |
| 速度 vs 速率 | 速度 = 位移/時間，速率 = 距離/時間 |
| 加速度 | a = Δv/Δt |
| 等加速運動 | 五大公式（v, v_0, a, t, d） |
| 自由落體 | a = g ≈ 9.8 m/s^2，與質量無關 |
| 拋體運動 | 水平等速 + 鉛直等加速，互相獨立 |
| 相對運動 | v_AB = v_A - v_B |

掌握這些概念後，你就具備了分析一切直線和曲線運動的基本工具。接下來學習牛頓運動定律時，就能將「力」與「加速度」連結起來，從「描述運動」進階到「解釋運動的原因」。
