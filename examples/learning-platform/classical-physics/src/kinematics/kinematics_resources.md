# 運動學：延伸資源

## 使用說明

以下資源依據類型分類，並標註了難度等級，幫助你根據自己的需求與背景選擇合適的學習材料。

**難度等級說明：**
- ⭐ 入門｜適合高中一年級或零基礎的學生
- ⭐⭐ 進階｜適合已掌握基本概念、想進一步練習的學生
- ⭐⭐⭐ 挑戰｜適合準備學測/指考或對物理特別有興趣的學生

## 書籍推薦

### 1.《觀念物理》（全套六冊）
- **作者：** Paul G. Hewitt 著，常雅珍 譯
- **出版資訊：** 天下文化出版
- **難度：** ⭐
- **推薦理由：** 這是全球最受歡迎的物理入門書之一。作者用淺白的文字、大量的生活例子和圖片來解釋物理觀念，幾乎不使用數學公式。第一冊「牛頓運動學·動量」直接對應運動學的所有核心概念。特別適合覺得物理課本太生硬、想要先建立直覺的同學。

### 2.《物理學的演進》
- **作者：** Albert Einstein、Leopold Infeld 合著，繁體中文譯本
- **出版資訊：** 商周出版
- **難度：** ⭐⭐
- **推薦理由：** 愛因斯坦親自撰寫的科普經典。書中以故事般的方式，從伽利略的運動學出發，帶領讀者走過物理學三百年的思想歷程。你會看到運動學如何自然地演變為牛頓力學，再發展為相對論。適合想要理解「這些公式背後的大故事」的好奇讀者。

### 3.《費曼物理學講義》第一卷
- **作者：** Richard P. Feynman
- **出版資訊：** 天下文化出版（中譯本）；英文原版可於 Caltech 網站免費閱讀
- **難度：** ⭐⭐⭐
- **推薦理由：** 諾貝爾物理學獎得主費曼的傳奇講義。第 8-10 章涵蓋運動學的核心內容，費曼的講解方式獨樹一格——他會用最意想不到的角度切入問題。這套書的數學程度較高（涉及微積分），適合程度優異且想要真正「理解」而非「背誦」的學生。英文好的同學推薦讀原文。

### 4. 高中物理課本與參考書
- **推薦版本：** 龍騰、翰林、南一等主要版本的高中物理課本
- **難度：** ⭐ 至 ⭐⭐
- **推薦理由：** 課本是最直接對應考試範圍的學習材料。建議搭配一本題解式參考書（如「大滿貫物理」、「搶救物理」等），先讀課本理解概念，再透過參考書大量練習。

### 5.《別鬧了，費曼先生》
- **作者：** Richard P. Feynman，吳程遠 譯
- **出版資訊：** 天下文化出版
- **難度：** ⭐
- **推薦理由：** 嚴格來說這不是物理教科書，而是費曼的自傳。但它能讓你感受到一個偉大物理學家如何思考問題、如何對世界保持好奇心。讀完之後你可能會覺得：「原來學物理可以這麼有趣。」很多人讀了這本書之後對物理產生興趣。

## 線上課程

### 1. 均一教育平台 — 高中物理：運動學
- **語言：** 繁體中文
- **難度：** ⭐
- **費用：** 完全免費
- **推薦理由：** 台灣本土的免費教育平台，內容完全對應台灣高中課綱。影片講解清晰，搭配即時練習題，可以一邊學一邊練。「運動學」單元涵蓋位移、速度、加速度、等加速運動和自由落體。特別適合需要課後複習或想要超前進度的同學。
- **連結：** https://www.junyiacademy.org/

### 2. Khan Academy — Physics: One-dimensional motion
- **語言：** 英文（部分有中文字幕）
- **難度：** ⭐ 至 ⭐⭐
- **費用：** 完全免費
- **推薦理由：** 全球最知名的免費教育平台之一。Khan Academy 的物理課程從最基礎開始，每個概念都有影片講解、文字教材和練習題。「One-dimensional motion」和「Two-dimensional motion」兩個單元完整覆蓋運動學。講解風格非常友善，不會讓人有壓力。英文程度中等以上的同學推薦使用。
- **連結：** https://www.khanacademy.org/science/physics

### 3. MIT OpenCourseWare — Classical Mechanics (8.01)
- **語言：** 英文
- **難度：** ⭐⭐⭐
- **費用：** 完全免費
- **推薦理由：** 麻省理工學院的開放課程，由 Walter Lewin 教授主講的版本尤為經典。Lewin 教授的物理演示（demonstration）舉世聞名——他會在課堂上實際做實驗來驗證理論。前幾堂課涵蓋運動學，Lewin 教授的教學風格生動有趣，但數學程度是大學等級。適合想要挑戰自己的學生。
- **連結：** https://ocw.mit.edu/courses/8-01sc-classical-mechanics-fall-2016/

### 4. Coursera — 力學：從日常到宇宙（國立臺灣大學）
- **語言：** 繁體中文
- **難度：** ⭐⭐
- **費用：** 可免費旁聽，證書需付費
- **推薦理由：** 台大物理系開設的線上課程，以中文授課，內容涵蓋古典力學的核心主題。課程從運動學開始，逐步推進到牛頓定律和能量。講解嚴謹但不失趣味，適合想要以大學水準系統學習力學的高中生。
- **連結：** 在 Coursera 搜尋「力學」或「mechanics NTU」

## 影片與 YouTube 頻道

### 1. 吳老師的物理教室（YouTube）
- **語言：** 繁體中文
- **難度：** ⭐ 至 ⭐⭐
- **推薦理由：** 台灣高中物理老師製作的教學影片，完全對應台灣課綱。講解仔細、例題豐富，特別適合跟不上學校進度、需要重新學一遍的同學。運動學相關的影片涵蓋了從基本定義到複雜題型。
- **連結：** 在 YouTube 搜尋「高中物理 運動學」

### 2. 3Blue1Brown — Essence of Calculus（微積分的本質）
- **語言：** 英文（有中文字幕）
- **難度：** ⭐⭐
- **推薦理由：** 雖然這是一個數學頻道而非物理頻道，但它用精美的動畫解釋微積分的核心概念——而微積分正是運動學背後的數學語言。看完「導數」和「積分」的幾集後，你會更深刻地理解為什麼「速度是位移的微分」、「位移是速度的積分」。視覺呈現非常出色。
- **連結：** https://www.youtube.com/c/3blue1brown

### 3. Veritasium — 物理科普影片
- **語言：** 英文（有中文字幕）
- **難度：** ⭐
- **推薦理由：** 澳洲物理學博士 Derek Muller 主持的科普頻道，擅長用實驗和街頭訪問來展示物理概念。其中「Which falls faster?」等影片直接涉及自由落體和運動學的核心概念。風格輕鬆有趣，適合建立物理直覺。
- **連結：** https://www.youtube.com/c/veritasium

### 4. PhET 模擬動畫（科羅拉多大學）
- **語言：** 多語言（含繁體中文）
- **難度：** ⭐
- **推薦理由：** 嚴格來說這不是影片，而是互動模擬器。PhET 提供了「拋體運動」（Projectile Motion）和「移動的人」（The Moving Man）等模擬，你可以自由調整參數，觀察物體的運動如何改變。對建立物理直覺非常有幫助。
- **連結：** https://phet.colorado.edu/zh_TW/simulations/filter?subjects=motion

## 網站與工具

### 1. PhET Interactive Simulations
- **類型：** 互動模擬工具
- **語言：** 繁體中文介面
- **難度：** ⭐
- **推薦理由：** 科羅拉多大學開發的免費物理模擬工具，運動學相關的模擬包括：「拋體運動」——可以調整角度和初速度看軌跡；「移動的人」——直觀展示位移、速度、加速度的關係和圖形意義。強烈建議在學習運動學時搭配使用。
- **連結：** https://phet.colorado.edu/zh_TW/

### 2. Desmos 圖形計算器
- **類型：** 線上圖形工具
- **語言：** 英文（介面直觀，不需太多英文能力）
- **難度：** ⭐ 至 ⭐⭐
- **推薦理由：** 免費的線上圖形計算器，可以繪製各種函數圖形。在學習運動學時，你可以用它畫出 x-t 圖、v-t 圖、a-t 圖，調整參數看圖形如何變化。也可以畫拋物線軌跡。對理解運動學的圖形分析非常有幫助。
- **連結：** https://www.desmos.com/calculator

### 3. HyperPhysics — Mechanics
- **類型：** 概念地圖式物理百科
- **語言：** 英文
- **難度：** ⭐⭐
- **推薦理由：** 喬治亞州立大學製作的物理學概念地圖網站。運動學的每個概念都有清晰的圖示和公式整理，概念之間用超連結相互關聯。適合在學習時快速查閱某個公式或概念的詳細說明。介面雖然看起來有點老舊，但內容非常扎實。
- **連結：** http://hyperphysics.phy-astr.gsu.edu/hbase/mechanics.html

### 4. Wolfram Alpha
- **類型：** 計算引擎
- **語言：** 英文
- **難度：** ⭐⭐
- **推薦理由：** 你可以直接輸入物理問題，例如「projectile motion initial velocity 30 m/s angle 45 degrees」，Wolfram Alpha 會自動計算射程、最高高度、飛行時間等。非常適合用來驗證你的手算答案是否正確。
- **連結：** https://www.wolframalpha.com/

### 5. GeoGebra
- **類型：** 互動幾何與物理模擬
- **語言：** 多語言（含繁體中文）
- **難度：** ⭐ 至 ⭐⭐⭐
- **推薦理由：** 免費的數學與物理互動工具，社群中有大量用戶分享的運動學模擬（搜尋 "kinematics" 或 "projectile motion"）。你也可以自己建立模擬。適合想要動手做、視覺化學習的同學。
- **連結：** https://www.geogebra.org/
