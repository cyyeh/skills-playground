# 雙軌擺盪模型：深度解析

> 本文深入探討 Stroebe 與 Schut 的**雙軌擺盪模型**（Dual Process Model, DPM），涵蓋其理論起源、核心機制、實證基礎與臨床應用。適合已具備基本悲傷心理學知識的讀者。

---

## 一、理論起源與發展背景

### 1.1 從階段模型到歷程模型

在 1990 年代以前，悲傷心理學的理論版圖主要由兩大取向主導：

- **階段模型**（Stage Models）：以 Kübler-Ross（1969）的五階段模型為代表，將悲傷描繪為一系列依序經歷的階段（否認、憤怒、討價還價、沮喪、接受）。
- **任務模型**（Task Models）：以 Worden（1991）的四項哀悼任務為代表，將悲傷視為需要主動完成的心理工作（接受失落現實、處理悲傷痛苦、適應新環境、重新定位與逝者的關係）。

然而，這些理論都面臨一個共同的問題：它們暗示悲傷是一個**單向的、線性的歷程**——人們要嘛在處理悲傷，要嘛已經「走出來」了。但臨床觀察與日常經驗都告訴我們，現實遠比這複雜得多。

### 1.2 DPM 的誕生

荷蘭烏特勒支大學的 **Margaret Stroebe** 與 **Henk Schut** 在觀察了大量喪親者的實際經驗後，於 **1999 年**正式提出了**雙軌擺盪模型**（Dual Process Model of Coping with Bereavement）。這個模型的核心洞見是：

> 悲傷因應不是一條直線，而是在兩種截然不同的因應取向之間**動態擺盪**的歷程。

Stroebe 與 Schut（1999）指出，先前的理論傾向於只強調悲傷因應的某一面——要嘛關注情緒的處理（如精神分析傳統中的「悲傷工作」），要嘛關注功能的恢復（如壓力因應模型）。DPM 則試圖整合這兩個面向，提出一個更貼近真實經驗的理論框架。

## 二、核心概念：雙軌與擺盪

### 2.1 失落導向因應（Loss-Oriented Coping, LO）

**失落導向因應**指的是直接面對與處理失落本身的心理歷程。這包括：

| 面向 | 具體表現 |
|------|----------|
| **思念與渴望** | 強烈渴望與逝者重聚，反覆回想共同經歷 |
| **悲傷工作** | 主動面對與處理失落帶來的痛苦情緒 |
| **反芻思考** | 反覆思索死亡的原因、意義與「如果」的假設 |
| **回憶** | 翻看照片、造訪有紀念意義的地方、整理遺物 |
| **哭泣與表達** | 透過哭泣、書寫或談論來表達悲傷 |
| **持續性連結** | 透過內在對話、紀念儀式等方式維持與逝者的情感聯繫 |

失落導向因應的功能在於**處理失落的情緒核心**——承認失落的現實，經歷悲傷的痛苦，逐漸將失落整合進個人的生命敘事之中。

### 2.2 復原導向因應（Restoration-Oriented Coping, RO）

**復原導向因應**指的是因應失落所帶來的**次級變化**——那些因為重要他人離去而必須面對的生活調整。這包括：

| 面向 | 具體表現 |
|------|----------|
| **新角色適應** | 學習逝者原本承擔的角色（如獨自管理財務、照顧孩子） |
| **身份重建** | 從「某人的配偶/父母」調整為新的自我認同 |
| **新關係建立** | 發展新的社交連結與支持網絡 |
| **生活重組** | 建立新的日常作息、調整居住安排 |
| **分心與迴避** | 暫時轉移注意力，給自己從悲傷中「喘息」的空間 |
| **否認或壓抑** | 暫時迴避悲傷的痛苦，以維持日常功能 |

值得注意的是，復原導向因應中的「迴避」與「分心」**並非病態表現**，而是適應歷程中的必要成分。Stroebe 與 Schut 特別強調，沒有人能夠持續不斷地沈浸在悲傷中而不崩潰——適度的迴避是心理健康的保護機制。

### 2.3 擺盪：模型的核心機制

**擺盪**（Oscillation）是 DPM 模型最獨特也最重要的概念。它指的是：

> 個體在失落導向與復原導向之間**自然地、動態地來回移動**，有時面對悲傷，有時迴避悲傷而專注於生活重建。

擺盪不是「選擇」的結果，而是一種**自發性的調節機制**。在同一天之內，一個人可能在上午整理逝者的衣物時淚流滿面（LO），下午卻能專注地處理工作事務（RO），晚上又因為看到某張照片而陷入思念（LO）。

**擺盪的特徵：**

- **動態性**：兩種因應之間的轉換是流動的，沒有固定的節奏
- **非自主性**：擺盪往往不受意識控制，觸發因素可能是外在的（如看到遺物）或內在的（如突然湧上的記憶）
- **適應性**：健康的擺盪讓個體既能處理悲傷，又不會被悲傷完全淹沒
- **時間變化**：隨著時間推移，擺盪的頻率與幅度通常會逐漸改變

```
【DPM 擺盪模型示意圖】

     失落導向 (LO)                     復原導向 (RO)
  ┌─────────────────┐              ┌─────────────────┐
  │ • 思念逝者      │              │ • 適應新角色    │
  │ • 悲傷工作      │   ◄──擺盪──► │ • 建立新身份    │
  │ • 回憶與哭泣    │              │ • 生活重組      │
  │ • 反芻思考      │              │ • 分心與休息    │
  └─────────────────┘              └─────────────────┘
            │                              │
            └──────── 日常生活經驗 ─────────┘
```

## 三、DPM 與其他理論的比較

### 3.1 與階段模型的比較

| 比較面向 | 階段模型（Kübler-Ross） | 雙軌擺盪模型（Stroebe & Schut） |
|----------|------------------------|---------------------------------|
| **悲傷歷程觀** | 線性、依序推進 | 動態、來回擺盪 |
| **因應重點** | 情緒階段的轉變 | 雙軌因應的平衡 |
| **個體差異** | 較少強調 | 明確容納不同的擺盪模式 |
| **迴避的角色** | 視為早期階段（否認） | 視為適應的必要成分 |
| **復原面向** | 較少涉及 | 明確納入為核心要素 |
| **實證支持** | 有限 | 較為充足 |

### 3.2 與任務模型的比較

Worden 的四項任務模型與 DPM 有一定的互補性。Worden 的前兩項任務（接受失落現實、處理悲傷痛苦）大致對應 LO 因應，後兩項任務（適應新環境、重新定位關係）則對應 RO 因應。然而，任務模型暗示這些任務有一定的先後順序，而 DPM 則強調**兩軌因應同時存在且交替進行**。

### 3.3 與 Bonanno 的韌性研究

George Bonanno（2004）的研究發現，面對喪親，約有 **50-60% 的人**展現出**韌性軌跡**（resilience trajectory）——他們從未出現顯著的功能障礙。從 DPM 的角度來看，這些韌性個體可能天生就具有較為靈活的擺盪能力，能夠在 LO 與 RO 之間流暢地轉換，既不過度沈溺於悲傷，也不完全迴避。

Bonanno 與 Kaltman（1999）進一步指出，**靈活性**（flexibility）是悲傷適應的關鍵預測因子——而這正是 DPM 中「健康擺盪」的核心特質。

## 四、實證基礎

### 4.1 支持性研究

DPM 自提出以來，已累積了相當的實證支持：

1. **日記研究**：Stroebe 與 Schut（2001）透過喪親者的每日日記追蹤，證實了 LO 與 RO 因應確實在日常生活中交替出現的模式。

2. **縱貫性研究**：Richardson（2010）的縱貫性研究發現，隨著時間推移，喪親者的因應模式確實從以 LO 為主逐漸轉向 LO 與 RO 的較均衡分配，支持了 DPM 的動態觀點。

3. **介入研究**：基於 DPM 設計的悲傷介入方案（同時處理 LO 與 RO 議題）比僅聚焦於某一面向的介入方案更為有效（Stroebe & Schut, 2010）。

4. **神經科學證據**：近年的腦造影研究發現，LO 與 RO 因應激活不同的神經網絡，支持了兩種因應確實涉及不同心理歷程的觀點（O'Connor, 2019）。

### 4.2 局限與批評

- **操作性定義**：LO 與 RO 的界線在某些情境下並不總是清晰，某些活動可能同時包含兩種因應的成分。
- **文化適用性**：模型主要基於西方（尤其是荷蘭）的研究資料，跨文化適用性有待更多驗證。
- **測量困難**：擺盪作為一個動態歷程，難以透過傳統的問卷或訪談在單一時間點進行測量。

## 五、性別差異與擺盪模式

### 5.1 研究發現

Stroebe 與 Schut（1999, 2015）特別關注了悲傷因應中的**性別差異**：

| 性別 | 典型傾向 | 潛在風險 |
|------|----------|----------|
| **女性** | 較傾向 LO 因應：更多情緒表達、思念、社交尋求支持 | 可能過度沈溺於悲傷，忽略生活重建 |
| **男性** | 較傾向 RO 因應：更多行動取向、問題解決、投入工作 | 可能迴避情緒處理，延遲悲傷反應 |

### 5.2 臨床啟示

DPM 的性別觀點帶來重要的臨床啟示：

- **對女性**：治療師可以在支持情緒表達的同時，適時引導關注復原導向的議題（如新角色適應、能力建立）。
- **對男性**：治療師可以在尊重其行動取向的同時，溫和地創造空間讓情緒得以表達與處理。
- **重要提醒**：這些是**統計上的傾向**，不是絕對的類別。個體差異遠大於性別差異，每個人的擺盪模式都是獨特的。

Doka 與 Martin（2010）提出的**直覺型與工具型悲傷風格**（Intuitive vs. Instrumental Grief Styles）與 DPM 的性別觀點高度呼應——直覺型（偏 LO）較常見於女性，工具型（偏 RO）較常見於男性，但兩種風格都不專屬於某一性別。

## 六、臨床應用

### 6.1 評估框架

DPM 為臨床評估提供了一個實用的框架：

**評估問題範例：**

- 個案目前的因應模式偏向哪一軌？（過度 LO 或過度 RO？）
- 個案是否能在兩種因應之間靈活擺盪？
- 擺盪的頻率與幅度是否隨時間有適當的變化？
- 是否有特定因素（文化、性別、社會支持）影響了擺盪模式？

### 6.2 介入策略

| 個案狀態 | 介入方向 | 具體策略 |
|----------|----------|----------|
| **過度 LO（沈溺於悲傷）** | 溫和引導向 RO | 漸進式活動安排、問題解決技巧、新角色練習 |
| **過度 RO（迴避悲傷）** | 溫和引導向 LO | 安全空間中的情緒探索、回憶整理、書寫練習 |
| **擺盪僵化** | 增進彈性 | 正念練習、情緒辨識訓練、接納承諾治療元素 |
| **擺盪混亂** | 建立結構 | 日常作息規劃、因應策略菜單、自我監測工具 |

### 6.3 團體治療的應用

在悲傷支持團體中，DPM 提供了一個共通的語言框架：

- 幫助成員理解自己與他人不同的因應模式
- 減少成員之間的比較與自我質疑（「為什麼我沒辦法像他/她那樣？」）
- 鼓勵成員接納擺盪的自然性，不強迫自己「持續悲傷」或「趕快好起來」

## 七、近期發展：修訂版 DPM

### 7.1 修訂版模型（Revised DPM）

Stroebe 與 Schut（2010, 2016）在原始模型的基礎上提出了修訂版，加入了幾個重要的新元素：

1. **過載與低載概念**（Overload/Underload）：
   - **過載**：個體在某一軌的因應需求超過其因應資源（如同時面對多重失落）
   - **低載**：個體在某一軌的因應投入不足（如過度迴避悲傷）

2. **正向與負向因素的區分**：
   - LO 因應中同時包含正向成分（溫暖的回憶、感恩）與負向成分（痛苦的思念、反芻）
   - RO 因應中同時包含正向成分（成就感、新可能性）與負向成分（角色壓力、孤獨）

3. **人際脈絡的強化**：修訂版更明確地將**社會支持**與**人際關係**納入模型，認為擺盪不只發生在個人內部，也發生在人際互動之中。

### 7.2 與意義重建的整合

Richardson（2010）及 Neimeyer（2001）的研究指出，**意義重建**（meaning reconstruction）可以被視為同時涉及 LO 與 RO 的歷程：在回顧失落的意義（LO）的同時，也在重新建構自我與世界的理解（RO）。這種整合觀點進一步豐富了 DPM 的理論深度。

### 7.3 跨文化應用的探索

近年來，研究者開始探索 DPM 在不同文化脈絡中的適用性。初步發現顯示：

- **集體主義文化**中，RO 因應可能更多地涉及家庭角色與社群義務的承擔
- **儀式豐富的文化**中，LO 與 RO 的擺盪可能更多地被文化儀式所結構化
- **表達限制的文化**中，LO 因應可能更多地以內在形式（而非外在表達）呈現

## 八、總結與展望

DPM 自 1999 年提出以來，已成為**當代悲傷心理學中最具影響力的理論之一**。它的核心貢獻在於：

1. **打破了線性迷思**：悲傷不是一條從痛苦到康復的直線，而是在面對與迴避之間的動態歷程
2. **正名了迴避的功能**：適度的迴避不是「沒有好好悲傷」，而是心理適應的必要成分
3. **容納了個體差異**：不同的擺盪模式反映了不同但同樣有效的因應方式
4. **提供了臨床地圖**：幫助治療師辨識個案的因應偏向，進而提供更精準的支持

展望未來，DPM 的研究方向將可能更多地與**神經科學**、**文化心理學**、**科技輔助介入**等領域整合，持續深化我們對悲傷這一人類共同經驗的理解。
