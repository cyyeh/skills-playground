# 術語表

> 悲傷心理學常用專業術語索引，按主題分類。點擊術語可在搜尋中找到相關內容。

## 基礎概念

| 術語 | English | 定義 |
|------|---------|------|
| 悲傷 | Grief | 個體因失落而產生的多維度情緒、認知、行為與生理反應的總和 |
| 哀悼 | Mourning | 悲傷的外在表達與社會行為，受文化規範與儀式所形塑 |
| 喪慟 | Bereavement | 因親近者死亡而經歷失落的客觀狀態 |
| 失落 | Loss | 失去有意義的人事物的經驗，包括死亡與非死亡性失落 |
| 預期性悲傷 | Anticipatory Grief | 在預期失落（如親人罹患末期疾病）尚未實際發生前即開始的悲傷反應 |
| 急性悲傷 | Acute Grief | 失落初期強烈的悲傷反應，包括強烈的思念、情緒波動與功能中斷 |
| 整合性悲傷 | Integrated Grief | 悲傷隨時間逐漸融入日常生活的狀態，失落的痛苦減弱但不消失 |
| 剝奪性悲傷 | Disenfranchised Grief | 不被社會認可或公開哀悼的悲傷，如流產、寵物死亡、分手等 |
| 悲傷反應 | Grief Reactions | 對失落的各種反應，涵蓋情緒、認知、行為與生理四個面向 |
| 適應性功能 | Adaptive Function | 悲傷在演化上幫助個體因應失落、調整依附關係與尋求社會支持的功能 |

## 理論模型

| 術語 | English | 定義 |
|------|---------|------|
| 悲傷五階段 | Five Stages of Grief | Kübler-Ross 提出的否認、憤怒、討價還價、沮喪與接受五個悲傷反應階段 |
| 否認 | Denial | 面對失落時的初始反應，難以相信或接受失落的現實 |
| 憤怒 | Anger | 對失落感到不公平、憤怒，可能指向自己、他人、逝者或更高力量 |
| 討價還價 | Bargaining | 試圖透過「如果...就好了」的假設思考來逆轉或緩解失落 |
| 沮喪 | Depression | 深刻體會到失落的不可逆性後產生的深層悲傷與空虛感 |
| 接受 | Acceptance | 逐漸承認失落的現實，不代表「沒事了」而是學習與失落共處 |
| 依附理論 | Attachment Theory | Bowlby 提出的理論，認為人類天生具有與重要他人建立情感連結的傾向 |
| 安全型依附 | Secure Attachment | 對自己與他人有正向看法，能在關係中感到安全，面對失落時悲傷但能復原 |
| 焦慮型依附 | Anxious Attachment | 對失去關係高度焦慮，面對失落時可能出現極度強烈且持久的悲傷反應 |
| 逃避型依附 | Avoidant Attachment | 傾向壓抑情緒、迴避親密，面對失落時可能表面平靜但延遲出現悲傷反應 |
| 混亂型依附 | Disorganized Attachment | 同時渴望與恐懼親密關係，面對失落時反應最為複雜且矛盾 |
| 雙軌擺盪模型 | Dual Process Model (DPM) | Stroebe 與 Schut 提出的理論，認為悲傷因應在失落導向與復原導向之間動態擺盪 |
| 失落導向 | Loss-Oriented | DPM 中面對失落本身的因應歷程，包括思念逝者、回憶、哭泣等 |
| 復原導向 | Restoration-Oriented | DPM 中面對失落帶來的次級變化的因應歷程，如學習新角色、建立新身份 |
| 擺盪 | Oscillation | DPM 的核心概念，指個體在失落導向與復原導向之間自然來回移動的歷程 |
| 悲傷任務模型 | Tasks of Mourning | Worden 提出的四項哀悼任務：接受失落現實、處理悲傷痛苦、適應新環境、重新定位關係 |
| 持續性連結 | Continuing Bonds | 喪親後與逝者維持內在情感連結的概念，現代觀點認為這有助於適應 |
| 意義重建 | Meaning Reconstruction | Neimeyer 提出的理論，強調悲傷者需要在失落經驗中重新建構意義 |

## 臨床診斷

| 術語 | English | 定義 |
|------|---------|------|
| 複雜性悲傷 | Complicated Grief | 悲傷反應在強度或持續時間上超出預期，嚴重影響日常功能的狀態 |
| 延長性悲傷障礙 | Prolonged Grief Disorder (PGD) | DSM-5-TR 與 ICD-11 中的正式診斷，指持續超過預期時間的強烈悲傷反應 |
| 慢性悲傷 | Chronic Grief | 悲傷反應長時間維持高強度而無明顯減緩的狀態 |
| 延遲性悲傷 | Delayed Grief | 失落後一段時間才出現明顯悲傷反應的模式 |
| 抑制性悲傷 | Inhibited Grief | 悲傷反應被壓抑或缺乏表達，可能以身體症狀或行為問題呈現 |
| 創傷性悲傷 | Traumatic Grief | 因突然、暴力或創傷性死亡導致的悲傷，常合併創傷後壓力反應 |
| 風險因子 | Risk Factors | 增加發展複雜性悲傷可能性的因素，如依附風格、死亡情境、社會支持等 |
| 保護因子 | Protective Factors | 降低複雜性悲傷風險的因素，如安全依附、社會支持、因應資源等 |
| 共病 | Comorbidity | 複雜性悲傷常與憂鬱症、PTSD、焦慮症等精神疾患同時出現 |

## 治療取向

| 術語 | English | 定義 |
|------|---------|------|
| 悲傷輔導 | Grief Counseling | 協助正常悲傷者完成哀悼任務的支持性介入 |
| 悲傷治療 | Grief Therapy | 針對複雜性悲傷提供的專業心理治療 |
| 意義重建治療 | Meaning Reconstruction Therapy | Neimeyer 發展的治療取向，幫助個案在失落中找到意義與成長 |
| 認知行為治療 | Cognitive Behavioral Therapy (CBT) | 透過改變與悲傷相關的不適應思考與行為模式來減輕症狀的治療取向 |
| 複雜性悲傷治療 | Complicated Grief Treatment (CGT) | Shear 發展的結構化治療方案，專門針對複雜性悲傷設計 |
| EMDR | Eye Movement Desensitization and Reprocessing | 眼動減敏與歷程更新治療，用於處理悲傷中的創傷成分 |
| 敘事治療 | Narrative Therapy | 透過重新敘說生命故事來重新框架失落經驗的治療取向 |
| 表達性藝術治療 | Expressive Arts Therapy | 運用藝術、音樂、舞蹈等創造性媒材來表達與處理悲傷的治療取向 |
| 支持性團體 | Support Groups | 由有類似失落經驗者組成的互助團體，提供同儕支持與正常化 |
| 哀悼儀式 | Mourning Rituals | 文化中用於標記失落、表達悲傷與促進社會支持的結構化行為 |

## 發展觀點

| 術語 | English | 定義 |
|------|---------|------|
| 死亡概念發展 | Development of Death Concept | 兒童隨年齡增長逐漸理解死亡的不可逆性、普遍性、無功能性與因果性 |
| 不可逆性 | Irreversibility | 理解死亡是永久的，死去的人不會再回來 |
| 普遍性 | Universality | 理解所有生物最終都會死亡，包括自己 |
| 無功能性 | Nonfunctionality | 理解死後身體所有功能完全停止 |
| 因果性 | Causality | 理解死亡有具體的生物學原因 |
| 回返悲傷 | Regrief | 兒童在不同發展階段以新的認知能力重新經歷與理解早年的失落 |
| 行為化 | Acting Out | 兒童透過行為問題（而非言語）表達悲傷的方式 |

## 文化與社會

| 術語 | English | 定義 |
|------|---------|------|
| 悲傷規則 | Grief Rules | 特定文化中關於誰可以悲傷、如何悲傷、悲傷多久的社會規範 |
| 集體悲傷 | Collective Grief | 整個社區或社會在共同經歷重大失落後的集體悲傷反應 |
| 文化腳本 | Cultural Scripts | 文化中預設的悲傷表達方式與哀悼行為模式 |
| 喪期 | Mourning Period | 文化規定的正式哀悼時間，如台灣的守喪、猶太教的 Shiva |
| 紀念儀式 | Memorial Rituals | 在特定時間點紀念逝者的文化實踐，如清明節、忌日、追思禮拜 |
| 跨文化悲傷 | Cross-Cultural Grief | 比較不同文化中悲傷表達、哀悼實踐與療癒觀點的研究領域 |
| 社會支持 | Social Support | 來自家庭、朋友、社區等的情感、資訊與實際協助 |
