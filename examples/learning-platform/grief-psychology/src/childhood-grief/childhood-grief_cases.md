# 兒童與青少年的悲傷：案例研究

> 每個孩子的悲傷都是獨特的，但透過觀察不同情境中的真實反應模式，我們可以更好地理解如何支持他們。以下案例均經過改編以保護隱私，但反映了臨床工作中常見的悲傷樣貌。

---

## 案例一：小雨的困惑——四歲女孩失去祖母

### 背景

小雨，四歲半，和爺爺奶奶同住。奶奶因肺癌在醫院過世。小雨最後一次見到奶奶是一週前，當時奶奶還能坐在床上和她說話。爸媽告訴她「奶奶去天上了」。

### 小雨的反應

小雨一開始看起來「好像沒事」。她繼續玩她的玩具，甚至對著大人笑。爸媽鬆了一口氣，以為她太小不懂。

然而接下來的幾天，小雨開始反覆問同樣的問題：
- 「奶奶在天上做什麼？她有帶衣服嗎？」
- 「天上冷不冷？她有東西吃嗎？」
- 「她什麼時候從天上回來？我要給她看我畫的圖。」

到了第二週，小雨開始出現**退化行為**——她已經不再尿床一年多了，卻連續好幾晚尿濕床單。晚上她堅持要開著燈、要媽媽握著她的手才肯睡。在幼兒園裡，老師反映小雨變得很「黏人」，不願意離開老師的視線。

有一天，小雨突然哭著說：「是不是因為我上次不乖，奶奶才走了？」

### 分析

小雨的反應典型地反映了**學齡前兒童**的悲傷特徵：

1. **暫時性思維**：小雨把死亡理解為暫時的「去了某個地方」，所以她關心奶奶在那裡的生活條件，並等待她回來。這符合 Nagy 描述的第一階段——將死亡視為可逆的分離。

2. **魔法思維與罪疚感**：小雨認為自己的行為（「不乖」）可能導致了奶奶的離開。這是 Piaget 所描述的前運思期**自我中心思考**的典型表現——孩子相信自己是世界的中心，因此也是事件的原因。

3. **水坑式悲傷**：她在悲傷與正常遊戲之間快速切換，並非因為不難過，而是這個年齡的孩子尚無法長時間維持強烈的情緒。

4. **退化行為與分離焦慮**：尿床、黏人、怕黑都是幼兒在面對失落時常見的退化反應，反映了安全感的動搖。

### 有效的支持方式

小雨的媽媽在諮詢兒童心理師後，做了以下調整：

- **修正解釋方式**：「奶奶的身體壞掉了，不能再修好了。她不能再呼吸、不能再吃東西、不能再走路了。奶奶已經死了，她不會回來了。但我們心裡面可以一直想著她。」
- **明確消除罪疚感**：蹲下來看著小雨的眼睛說：「奶奶的離開完全不是你的錯。不管你乖不乖，都和奶奶的身體壞掉沒有關係。奶奶非常非常愛你。」——這句話，她在接下來的幾週裡重複說了很多次。
- **透過遊戲表達**：提供玩偶讓小雨「演出」奶奶的故事。小雨反覆讓一個玩偶「去醫院」然後「不回來」，在遊戲中處理她的困惑。
- **建立紀念儀式**：和小雨一起在家裡放了一張奶奶的照片，每天晚上睡前可以「跟奶奶說晚安」。
- **維持日常穩定**：保持固定的幼兒園接送時間和睡前儀式，給予可預測的安全感。

## 案例二：阿凱的沉默——八歲男孩失去父親

### 背景

阿凱，八歲，就讀國小二年級。父親在一場交通事故中突然過世。阿凱和爸爸的關係很好，爸爸每個週末都會帶他去打棒球。媽媽在喪夫後陷入嚴重的悲傷，經常在家裡哭泣。

### 阿凱的反應

葬禮上，阿凱表現得異常安靜。他沒有哭，也沒有問任何問題。親戚們稱讚他「真懂事」、「真勇敢」。

回到學校後，阿凱的行為開始出現變化：
- **成績急遽下滑**：原本名列前茅的他，連續幾次考試不及格。老師發現他上課時經常發呆，叫他名字時會嚇一跳。
- **攻擊行為**：他開始在下課時與同學發生衝突。有一次因為同學無意間提到「我爸說……」，阿凱突然推倒了那個同學。
- **扮演照顧者**：在家裡，他開始主動做家事、倒垃圾、試圖安慰媽媽。他曾對阿姨說：「我現在是家裡的男生了，要照顧媽媽。」
- **身體症狀**：頻繁喊肚子痛，要求請假不上學。帶去看醫生，檢查結果一切正常。

最讓人心疼的一幕發生在父親過世兩個月後的某天晚上。媽媽聽到阿凱房間有聲音，推門進去發現他躲在棉被裡，一邊哭一邊對著爸爸的棒球手套說話：「爸爸，你為什麼不回來……你答應要教我投變化球的……」

### 分析

阿凱的案例展現了**學齡兒童**悲傷的幾個重要面向：

1. **情緒壓抑與角色逆轉**：阿凱察覺到媽媽的脆弱，於是壓抑自己的悲傷，試圖承擔「保護者」的角色。Christ（2000）的研究指出，9-11 歲的兒童特別容易出現這種**親職化**（parentification）現象，但阿凱在 8 歲就展現了這個傾向。

2. **悲傷的行為化表達**：在學校的攻擊行為是他無法用語言表達的憤怒與痛苦的外在展現。他不是「變壞了」，而是悲傷沒有出口。

3. **突然性死亡的衝擊**：父親的意外死亡沒有給家人任何心理準備的時間。突然性失落常導致更強烈的否認、憤怒與安全感崩塌。

4. **身體化症狀**：反覆的腹痛是悲傷以身體語言呈現的典型方式——身體在替心理「說」它說不出口的痛。

5. **私密的悲傷時刻**：阿凱對著棒球手套說話的行為，顯示他並非「不難過」，而是選擇在私密的空間中與逝去的父親維持**持續性連結**（continuing bonds）。

### 有效的支持方式

阿凱的家庭在學校輔導老師的建議下，採取了以下措施：

- **解除「小大人」的重擔**：媽媽在自己稍微穩定後，明確告訴阿凱：「照顧這個家是媽媽的工作，不是你的。你只需要當一個八歲的小孩。你可以難過，可以哭，可以生氣。」
- **提供替代的表達管道**：輔導老師引導阿凱用「寫信給爸爸」的方式表達感受。阿凱每週寫一封信，內容從生氣（「你為什麼丟下我」）到想念（「我今天棒球打得很好，好想讓你看到」）。
- **學校層面的支持**：老師給予學業上的彈性，暫時不因成績施壓。同時，私下告訴阿凱：「如果你在教室裡覺得很難受，可以舉手告訴我，我會讓你去輔導室休息一下。」
- **正常化攻擊行為背後的情緒**：「你生氣是因為你很想爸爸，這是可以理解的。但推人會傷到別人，下次你覺得很生氣的時候，可以去打沙袋或者跑操場。」
- **家庭諮商**：媽媽和阿凱一起參加了家庭悲傷諮商，學習彼此支持而不互相壓抑。

## 案例三：心柔的風暴——十五歲少女失去摯友

### 背景

心柔，十五歲，高一學生。她最好的朋友佳慧在一次意外墜樓中身亡。佳慧在事發前曾傳訊息給心柔說「我好累」，心柔當時正在念書，只回了「加油」兩個字。後來她得知佳慧的死可能是自殺。

### 心柔的反應

- **極度的罪疚感**：心柔反覆翻看那則訊息對話，不斷告訴自己：「如果我那時候多問一句，她是不是就不會死？」她把佳慧的最後一則訊息截圖設為手機桌布，彷彿在不斷懲罰自己。
- **社群媒體上的公開悲傷**：心柔在 Instagram 上連續發了好幾篇紀念佳慧的貼文，引發了大量同學的回應。有些留言帶給她安慰，但也有人私訊說「你是不是在消費她的死」，讓心柔更加崩潰。
- **存在性質疑**：她開始質疑生命的意義，對未來失去動力。跟媽媽說：「人隨時都可能死，念書有什麼意義？」
- **風險行為**：她的作息完全混亂，凌晨三四點才睡，有時候跟不太熟的學長姐出去喝酒。
- **憤怒**：對佳慧的父母感到憤怒（「他們怎麼都沒發現？」），也對佳慧本人感到憤怒（「你怎麼可以就這樣走了？」），然後對自己的憤怒感到罪疚。

### 分析

心柔的案例凸顯了**青少年悲傷**的獨特挑戰：

1. **同儕死亡的存在衝擊**：同齡人的死亡打破了青少年的「個人不朽神話」，強迫他們提前面對存在的脆弱性。

2. **可能是自殺的額外負擔**：當死亡原因可能涉及自殺時，存活的朋友常經歷更強烈的**罪疚感**、**憤怒**與**困惑**。心柔的「如果我多問一句」是典型的**反事實思考**（counterfactual thinking）。

3. **數位時代的悲傷**：社群媒體既是表達悲傷的平台，也是壓力的來源。網路上的公開悲傷可能招來誤解或批評，加重負擔。

4. **風險行為**：酒精使用、作息混亂等風險行為是青少年處理無法承受的情緒時常見的不健康因應方式。

5. **複雜的憤怒**：對逝者感到憤怒是自殺喪慟中特別常見的反應，但青少年常因為「對死人生氣」而感到深深的罪疚。

### 有效的支持方式

- **正常化複雜的感受**：學校輔導老師告訴心柔：「對佳慧感到生氣是完全可以理解的。你可以同時愛她、想她，又對她的選擇感到生氣。這些感受不矛盾。」
- **處理罪疚感**：「你那天回了『加油』，是因為你不知道她正在那麼痛苦。你不是心理醫生，你不可能從一則訊息就知道會發生什麼。這不是你的錯。」
- **同儕支持小組**：學校安排了悲傷支持小團體，讓佳慧的幾位好友一起分享感受，彼此理解。
- **關注自殺風險**：考量到心柔可能出現的模仿風險，諮商師定期評估她的安全狀態，並與家長保持密切溝通。
- **引導健康的數位悲傷**：諮商師與心柔討論了在社群媒體上紀念朋友的健康方式，同時也討論了何時需要暫時離開網路。
- **轉介專業諮商**：因為死亡原因的特殊性與心柔反應的強度，學校將她轉介給專門處理青少年創傷性悲傷的心理師。

## 案例四：小恩的隱形悲傷——十歲男孩失去哥哥

### 背景

小恩，十歲，哥哥小傑（十三歲）因白血病過世。小傑在確診後與病魔搏鬥了兩年，期間父母幾乎所有的心力都投注在照顧小傑上。小恩在這兩年間經常被寄放在阿姨家，他對此從不抱怨。

### 小恩的反應

小傑過世後，所有人的關注都在父母身上。親友來弔唁時會對父母說「節哀」，但很少有人問小恩「你還好嗎？」。

- **被忽略的悲傷**：小恩在學校寫了一篇作文《我的家人》，裡面完全沒有提到哥哥。老師覺得奇怪，問他為什麼，他說：「提到哥哥會讓爸爸媽媽難過，所以我不要寫。」
- **存活者罪疚感**：小恩偷偷問阿姨：「是不是因為我以前有時候不喜歡哥哥，所以老天才帶走他？」他想起自己曾因為父母都在照顧哥哥而在心裡想過「真希望沒有哥哥」，這個念頭讓他充滿自責。
- **矛盾的情緒**：他同時感到悲傷（失去了一起長大的玩伴）與如釋重負（父母終於不用一直在醫院了，可以陪他了），而「如釋重負」的感覺又讓他覺得自己是個壞人。
- **過度順從**：小恩變得極度「乖巧」，從不頂嘴、從不提出要求。他似乎在試圖成為「完美的孩子」，以免再讓父母失去什麼。

### 分析

小恩是典型的「**被遺忘的悲傷者**」（forgotten mourner）——在手足死亡的家庭中，存活的兒童常被忽略。

1. **被剝奪的悲傷權**：所有人的焦點都在失去孩子的父母身上，小恩的失落被無意間邊緣化了。他的悲傷屬於一種**剝奪性悲傷**（disenfranchised grief）的變體。

2. **存活者罪疚感的獨特表現**：手足間本來就會有競爭、嫉妒與衝突的正常情緒。但當手足過世，這些再正常不過的念頭都可能變成沉重的罪疚來源。

3. **矛盾情緒的正常性**：同時感到悲傷與解脫，是長期照顧末期患者的家屬（包括兒童）非常常見的反應，被稱為**矛盾性悲傷**（ambivalent grief）。

4. **補償行為**：過度順從是小恩試圖「彌補」的方式——彌補他以為自己造成的傷害，也避免給父母增添更多痛苦。

### 有效的支持方式

小恩的阿姨在觀察到這些跡象後，主動與小恩的父母溝通，促成了以下改變：

- **看見小恩**：父母在自己的悲傷稍微穩定後，特別撥出時間和小恩單獨相處，告訴他：「失去哥哥，你也很難過，對不對？爸媽以前花太多時間在醫院，讓你一個人，我們很抱歉。我們也很想知道你的感受。」
- **正常化矛盾情緒**：「你有時候覺得鬆了一口氣，是因為這兩年真的太辛苦了。這不代表你不愛哥哥。你可以同時很想他，又覺得輕鬆了一點，這兩種感覺都是真的。」
- **處理罪疚感**：「你以前偶爾不喜歡哥哥，每個小孩對自己的兄弟姊妹都會有這種時候。這些想法完全不會讓人生病或死掉。哥哥是因為白血病才離開的，跟你完全沒有關係。」
- **允許提起哥哥**：全家開始在餐桌上自然地談起小傑——好笑的回憶、一起做過的事。小恩第一次在家人面前為哥哥哭了出來。
- **手足悲傷支持團體**：小恩參加了一個專門為失去手足的兒童設計的支持團體，在那裡他遇到了其他有類似經歷的孩子，第一次感到「有人懂我」。

## 案例五：安安的康復之路——六歲女孩失去媽媽與積極的成人支持

### 背景

安安，六歲，媽媽因乳癌過世。在媽媽確診後的一年治療期間，爸爸和外婆共同成為安安的主要照顧者。爸爸是一位國小老師，對兒童發展有基本的認識。在媽媽病情惡化時，他就開始預先做準備——閱讀相關書籍、諮詢學校的心理師、並與外婆商量如何在媽媽離開後支持安安。

### 爸爸的預備與應對

#### 在媽媽離世前

- **誠實但適齡的溝通**：爸爸在媽媽病情加重時，告訴安安：「媽媽身體裡有壞細胞，醫生一直在想辦法把壞細胞趕走。但有時候壞細胞太厲害了，身體就會越來越虛弱。」
- **提供參與機會**：讓安安畫卡片給媽媽、在醫院陪媽媽時唱歌、幫媽媽塗護手霜。這些參與讓安安感到自己有能力做些什麼。
- **預告可能的結果**：「醫生說媽媽的身體可能沒辦法好起來了。如果媽媽的身體完全壞掉了，她就會死。這不是任何人的錯。」

#### 在媽媽離世後

- **清楚的告知**：爸爸抱著安安，平靜但悲傷地說：「安安，媽媽今天早上走了。她的身體壞掉了，不能再修好了。媽媽死了。爸爸很傷心，我知道你也會很傷心。」
- **允許情緒流動**：安安先是困惑，然後大哭，然後問「那今天還可以看卡通嗎？」爸爸沒有對任何反應感到驚訝或不適，因為他知道這就是六歲孩子的悲傷節奏。
- **讓安安選擇參與告別式的方式**：「你可以去跟媽媽說再見，也可以不去。如果你想去，你可以帶一樣東西放在媽媽旁邊。如果中間你不想待了，外婆會陪你出去。」安安選擇帶了自己畫的全家福。

### 安安的悲傷歷程

- **初期**（前兩週）：交替出現哭泣與正常遊戲。反覆問「媽媽現在在哪裡？」，爸爸每次都耐心回答。
- **第一個月**：出現輕微的退化行為（要求爸爸餵她吃飯），分離焦慮增加（上學時不肯放開爸爸的手）。爸爸配合她的需求，但也溫柔地提供過渡物品（媽媽的一條圍巾，讓安安帶在身邊）。
- **三個月後**：安安在幼兒園的「我的家庭」主題中，畫了天上有一個笑臉，說「那是媽媽在看我」。她開始能自然地提起媽媽，而不總是伴隨哭泣。
- **六個月後**：安安大致恢復了正常的作息與社交。偶爾會在睡前說「想媽媽」，爸爸會陪她看媽媽的照片、說說關於媽媽的故事。
- **一年後**：在媽媽的忌日，全家人一起去媽媽最喜歡的海邊，放了一個風箏。安安說：「媽媽一定看到了。」

### 分析：為什麼安安的適應相對良好？

安安的案例展示了**積極的成人支持**如何成為兒童悲傷適應中最關鍵的保護因子：

1. **預備性溝通**：在失落發生前就開始適齡的溝通，讓安安不會在死亡發生時完全被震驚擊垮。

2. **穩定的照顧者**：爸爸雖然自己也在悲傷，但他有意識地維持了自己作為照顧者的功能。Christ（2000）的研究反覆強調：存活照顧者的功能狀態是兒童適應的最佳預測因子。

3. **開放的溝通文化**：家中沒有將死亡和悲傷視為禁忌話題。安安可以自由地提問、表達情緒、提起媽媽。

4. **維持日常結構**：在情感上給予彈性的同時，生活的基本結構被維持，提供了穩定感。

5. **持續的紀念**：透過照片、故事、紀念儀式，安安被允許與媽媽維持**持續性連結**，而不是被要求「忘掉」或「走出來」。

6. **支持網絡**：爸爸不是孤軍奮戰——外婆的協助、學校心理師的諮詢、爸爸自己的閱讀與學習，都構成了重要的支持系統。

> **核心啟示：** 兒童不需要完美的大人，但需要一個夠穩定、夠真誠、願意陪伴的大人。安安的爸爸並非沒有悲傷或不犯錯，但他的準備、開放與穩定，為安安鋪出了一條相對順暢的悲傷之路。

## 案例反思：跨案例的共同主題

| 主題 | 案例中的呈現 |
|------|------------|
| **發展階段決定悲傷表達方式** | 小雨的反覆提問 vs. 阿凱的行為化 vs. 心柔的存在性質疑 |
| **罪疚感跨越所有年齡** | 小雨的魔法思維罪疚、阿凱的隱性自責、心柔的「如果我多問一句」、小恩的存活者罪疚 |
| **成人支持是最關鍵的因子** | 安安案例最明確展現，但每個案例中有效的介入都來自穩定的成人 |
| **悲傷常被「看見」的形式不是哭泣** | 腹痛、攻擊行為、過度順從、風險行為——這些都是悲傷的「語言」 |
| **每個孩子都需要被允許用自己的方式悲傷** | 沒有「正確的」悲傷方式，但每個孩子都需要被看見、被理解、被陪伴 |
