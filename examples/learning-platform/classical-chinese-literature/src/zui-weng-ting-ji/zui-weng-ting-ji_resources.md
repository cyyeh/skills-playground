# 醉翁亭記：延伸學習資源

> 以下資源依照難度分級：
> - 初階（適合初次接觸文言文的國中生）
> - 中階（適合有一定文言文基礎的高中生）
> - 進階（適合想深入研究的進階學習者）

---

## 書籍推薦

### 1. 古文觀止
- **編者：** 吳楚材、吳調侯（清代）
- **難度：** 中階
- **推薦理由：** 收錄了自先秦至明末的兩百餘篇經典古文，《醉翁亭記》即在其中。每篇附有註釋和評點，是學習文言文最重要的入門選集。建議選擇附有白話翻譯的現代註解版，方便對照理解。同時收錄了《岳陽樓記》，適合兩篇對照閱讀。
- **購買連結：** [博客來——古文觀止（全譯本）](https://www.books.com.tw/products/0010825498)

### 2. 唐宋八大家文鈔
- **編者：** 茅坤（明代）
- **難度：** 中階至進階
- **推薦理由：** 收錄唐宋八大家的代表作品，歐陽修作為八大家之一，其散文在書中佔有重要篇幅。閱讀歐陽修的其他作品（如《秋聲賦》《朋黨論》《瀧岡阡表》），可以更全面地理解他的文學風格和思想內涵。
- **購買連結：** [博客來——唐宋八大家文鈔](https://www.books.com.tw/web/sys_serialtext/?item=0010547817)

### 3. 歐陽修詩文集
- **作者：** 歐陽修；洪本健校箋
- **難度：** 進階
- **推薦理由：** 歐陽修全部詩文作品的校注本，是研究歐陽修文學成就的權威版本。除了《醉翁亭記》，還可以閱讀《秋聲賦》《豐樂亭記》等作品，了解歐陽修在滁州時期的完整文學創作。
- **購買連結：** [博客來——歐陽修詩文集](https://www.books.com.tw/products/CN11035271)

### 4. 中國古典散文鑑賞
- **作者：** 張高評 等
- **難度：** 中階
- **推薦理由：** 台灣學者編寫的古典散文賞析專書，以深入淺出的方式解讀歷代經典散文。其中對記體散文（包括《醉翁亭記》《岳陽樓記》等）有精到的分析，適合想進一步理解散文藝術的高中生和大學生。
- **購買連結：** [博客來——中國古典散文鑑賞](https://www.books.com.tw/products/0010547156)

### 5. 宋代文學史
- **作者：** 孫望、常國武 主編
- **難度：** 進階
- **推薦理由：** 系統性地介紹宋代文學的發展脈絡，包含歐陽修在古文運動中的領袖地位、宋代散文的藝術特色等內容。適合想從文學史的宏觀角度理解《醉翁亭記》的進階學習者。
- **購買連結：** [博客來——宋代文學史](https://www.books.com.tw/products/CN11784523)

## 線上課程

- **平台：** 均一教育平台
- **難度：** 初階
- **語言：** 繁體中文
- **費用：** 免費
- **網址：** [https://www.junyiacademy.org/](https://www.junyiacademy.org/)
- **推薦理由：** 台灣本土的免費教育平台，內容對應國中課綱。有文言文篇章的影片教學和練習題，適合正在學習《醉翁亭記》的國中生做課後複習。互動式練習可以幫助鞏固對重要字詞和句式的理解。

### 2. 翰林雲端學院——高中國文
- **平台：** 翰林出版
- **難度：** 中階
- **語言：** 繁體中文
- **費用：** 部分免費，完整版需付費
- **網址：** [https://www.ehanlin.com.tw/](https://www.ehanlin.com.tw/)
- **推薦理由：** 提供高中國文課程的線上教學影片，包含《醉翁亭記》的詳細講解、修辭分析和考試重點整理。特別適合準備學測和指考的高中生，有系統化的考題練習。

### 3. 國家教育研究院——愛學網
- **平台：** 國家教育研究院
- **難度：** 初階至中階
- **語言：** 繁體中文
- **費用：** 免費
- **網址：** [https://stv.naer.edu.tw/](https://stv.naer.edu.tw/)
- **推薦理由：** 提供配合台灣國中小課程的教學影片，包含文言文篇章的基礎導讀。影片由現職教師錄製，講解清晰易懂，適合需要基礎引導的學生。

### 4. Coursera — 中國哲學與文化相關課程
- **平台：** Coursera
- **難度：** 進階
- **語言：** 英文
- **費用：** 可免費旁聽
- **網址：** [https://www.coursera.org/search?query=chinese+literature+culture](https://www.coursera.org/search?query=chinese+literature+culture)
- **推薦理由：** Coursera 上有多門探討中國文學與文化的課程，可搜尋 "Chinese literature" 或 "Chinese culture" 找到相關內容。這些課程對理解宋代文人的精神世界、士大夫的社會角色等背景知識非常有幫助。適合想用英文視角理解中國文學的學習者。

### 2. 呂捷歷史——宋代文學與政治
- **平台：** YouTube

- **語言：** 繁體中文
- **網址：** [https://www.youtube.com/@ljhistory](https://www.youtube.com/@ljhistory)

### 4. CrashCourse — Chinese History (英文)
- **平台：** YouTube

- **語言：** 英文

## 網站與工具

### 1. 中國哲學書電子化計劃（CTEXT）
- **難度：** 中階至進階
- **語言：** 繁體中文 / 英文
- **網址：** [https://ctext.org/zh](https://ctext.org/zh)
- **推薦理由：** 收錄了大量中國古典文獻的全文檢索資料庫，可以查閱歐陽修的其他作品以及《論語》《孟子》等與「醉翁亭記」思想相關的原典。支持全文搜索和文本對照，是研究古文的必備工具。

### 2. 教育部重編國語辭典修訂本
- **難度：** 初階至進階
- **語言：** 繁體中文
- **網址：** [https://dict.revised.moe.edu.tw/](https://dict.revised.moe.edu.tw/)
- **推薦理由：** 台灣教育部官方的權威辭典，可以查詢文言文詞彙的詳細釋義和用法。學習《醉翁亭記》時遇到「蔚然」「頹然」「觥籌」等不懂的字詞，這是最可靠的查詢工具。

### 3. 漢典
- **難度：** 初階至中階
- **語言：** 繁體中文 / 簡體中文
- **網址：** [https://www.zdic.net/](https://www.zdic.net/)
- **推薦理由：** 提供漢字的詳細解釋，包括字源演變、讀音、用法等。對於理解文言文中的單字意義（如「翼」「輒」「洌」「翳」等）特別有幫助。介面簡潔易用。

### 4. 全宋詩文檢索系統
- **難度：** 中階至進階
- **語言：** 繁體中文
- **網址：** [https://cls.lib.ntu.edu.tw/song/](https://cls.lib.ntu.edu.tw/song/)
- **推薦理由：** 台灣大學中國文學系建置的宋代詩文全文檢索系統，可以搜索歐陽修的全部詩文作品，包括他在滁州時期寫下的其他文章（如《豐樂亭記》）。適合想深入了解歐陽修文學全貌的進階學習者。

### 5. Pleco Chinese Dictionary App (英文/中文)
- **難度：** 初階至中階
- **語言：** 英文 / 中文
- **網址：** [https://www.pleco.com/](https://www.pleco.com/)
- **推薦理由：** 功能強大的中文字典 App，支持手寫辨識和 OCR 文字識別。閱讀文言文時遇到不認識的字，可以直接手寫查詢或拍照辨識。內建多部權威辭典，包括文言文專用辭典。適合中英雙語學習者。
