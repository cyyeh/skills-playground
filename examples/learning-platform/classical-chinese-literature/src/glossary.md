# 詞彙表

## 常見虛詞

| 詞彙 | English | 定義 |
|------|---------|------|
| 之 | zhī (structural particle) | 結構助詞，相當於「的」；也可作代詞「他/她/它」或動詞「到、往」 |
| 乎 | hū (question particle) | 語氣詞，表疑問「嗎」或感嘆「啊」；也可作介詞「在、於」 |
| 者 | zhě (nominalizer) | 助詞，表「……的人/事物」；也用於判斷句 |
| 也 | yě (declarative particle) | 語氣詞，表判斷或肯定；醉翁亭記中大量使用，形成獨特韻律 |
| 矣 | yǐ (completion particle) | 語氣詞，表示事情已然完成或感嘆，相當於「了」 |
| 焉 | yān (particle/pronoun) | 兼詞，相當於「於此」；也作語氣詞 |
| 其 | qí (possessive/modal) | 代詞「他的/那個」；也作語氣副詞，表推測、反問或期望 |
| 而 | ér (conjunction) | 連接詞，可表順承「然後」、轉折「但是」、並列「又」等多種關係 |
| 則 | zé (then/however) | 連詞，表承接「就」或轉折「卻」 |
| 於 | yú (preposition) | 介詞，相當於「在、到、對於、比」等 |
| 以 | yǐ (by means of) | 介詞「用、拿、憑」；連詞「因為、來」 |
| 為 | wéi/wèi (be/for) | 動詞「做、當、是」(wéi)；介詞「為了、替」(wèi) |
| 所 | suǒ (that which) | 與動詞組成「所＋動詞」結構，表被動或名詞化 |
| 乃 | nǎi (thereupon) | 副詞「於是、才、竟然」；判斷詞「是」 |
| 遂 | suì (thereupon) | 副詞，表示「於是、就」，常用於敘事文中承接上文 |

## 常見實詞

| 詞彙 | English | 定義 |
|------|---------|------|
| 謂 | wèi (say/call) | 動詞，「說、稱呼、認為」 |
| 云 | yún (say) | 動詞，「說」，常見於引述他人言語 |
| 觀 | guān (observe) | 動詞，「觀看、觀察」 |
| 居 | jū (dwell) | 動詞，「居住、處於」 |
| 往 | wǎng (go) | 動詞，「前往」 |
| 至 | zhì (arrive) | 動詞，「到達」；形容詞「最、極」 |
| 屬 | shǔ/zhǔ (belong/entrust) | 動詞，「屬於、類」(shǔ)；「囑咐、連綴」(zhǔ) |
| 勝 | shèng (surpass) | 動詞，「勝過、承受」；名詞「美景」 |
| 臨 | lín (face/approach) | 動詞，「面對、靠近、到來」 |
| 遷 | qiān (move/demote) | 動詞，「遷移、升遷、貶謫」（左遷＝貶官） |
| 諫 | jiàn (remonstrate) | 動詞，「直言規勸君主或長輩」 |
| 志 | zhì (will/aspiration) | 名詞，「志向、心意」；動詞「記錄」 |
| 道 | dào (way/principle) | 名詞，「道路、道理、學說」；動詞「說」 |
| 賢 | xián (virtuous) | 形容詞，「有才德的」；名詞「賢人」 |
| 鴻儒 | hóngrú (great scholar) | 名詞，「博學的大儒」，見於陋室銘 |

## 文體術語

| 詞彙 | English | 定義 |
|------|---------|------|
| 銘 | míng (inscription) | 文體名，刻在器物上用以警戒或記述功德的韻文，如陋室銘 |
| 說 | shuō (discourse) | 文體名，闡述道理的短文，如愛蓮說、師說 |
| 記 | jì (record/essay) | 文體名，記錄見聞或抒發感想的散文，如桃花源記、岳陽樓記 |
| 表 | biǎo (memorial) | 文體名，臣子向君主上呈的奏議文書 |
| 賦 | fù (rhyme-prose) | 文體名，介於詩歌與散文之間的韻文 |
| 樂府 | yuèfǔ (Music Bureau poetry) | 漢代官署名，後泛指民間歌謠體裁的詩歌，如木蘭詩 |
| 古文 | gǔwén (classical prose) | 指先秦兩漢的散文體裁，韓愈倡導的古文運動即以此為典範 |
| 駢文 | piánwén (parallel prose) | 講究對仗、聲律的文體，古文運動的對立面 |

## 修辭術語

| 詞彙 | English | 定義 |
|------|---------|------|
| 對偶 | duìǒu (parallelism) | 用結構相同、字數相等的語句兩兩對應，如「苔痕上階綠，草色入簾青」 |
| 排比 | páibǐ (serial parallelism) | 三個以上結構相似的語句連用，加強語勢 |
| 比興 | bǐxīng (metaphor & evocation) | 先以其他事物起興，再引出要表達的主題，源自《詩經》 |
| 託物言志 | tuōwù yánzhì (expressing ideals through objects) | 借描寫外物來寄寓作者志向或品格，如以蓮花喻君子 |
| 借代 | jièdài (metonymy) | 用相關事物代替本體，如「絲竹」代指音樂 |
| 誇飾 | kuāshì (hyperbole) | 故意誇大或縮小事實以增強效果，如「銜遠山，吞長江」形容洞庭湖的壯闊 |
| 互文 | hùwén (interlocking) | 上下文互相補充、合併理解，如「將軍百戰死，壯士十年歸」 |
| 用典 | yòngdiǎn (literary allusion) | 引用歷史故事或前人詞句來豐富文意 |
| 設問 | shèwèn (rhetorical question) | 自問自答以引起讀者注意 |
| 反問 | fǎnwèn (rhetorical question) | 用疑問語氣表達確定意思，加強語勢 |

## 文化與歷史術語

| 詞彙 | English | 定義 |
|------|---------|------|
| 古文運動 | gǔwén yùndòng (Classical Prose Movement) | 唐代韓愈、柳宗元倡導，主張以先秦散文取代華麗的駢文 |
| 唐宋八大家 | Táng-Sòng bā dàjiā (Eight Great Prose Masters) | 韓愈、柳宗元、歐陽修、蘇洵、蘇軾、蘇轍、王安石、曾鞏 |
| 士 | shì (scholar-official) | 古代讀書人，兼有知識分子與官員的身份 |
| 隱逸 | yǐnyì (reclusion) | 退隱山林、不求仕進的生活態度，陶淵明是代表人物 |
| 貶謫 | biǎnzhé (demotion/exile) | 古代官員因罪或政治原因被降職外放 |
| 先憂後樂 | xiānyōu hòulè (worry first, enjoy later) | 范仲淹「先天下之憂而憂，後天下之樂而樂」的精神 |
| 桃花源 | táohuāyuán (Peach Blossom Spring) | 陶淵明筆下的理想世界，後泛指世外桃源 |
