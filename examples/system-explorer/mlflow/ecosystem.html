Community & Governance

🏛
Linux Foundation project since 2024, ensuring vendor-neutral governance. MLflow is backed by Databricks but governed by an open community. Regular monthly releases, active GitHub Discussions, Slack channel, and Stack Overflow community. Plugin ecosystem enables third-party extensions for custom stores, deployers, and evaluators.
