Language Support

Python is the primary SDK with the richest feature set. TypeScript/JavaScript provides auto-tracing for OpenAI and Gemini. Java/Scala enables JVM-based tracking and Spark integration. R provides experiment tracking from R workflows. The REST API serves as the language-agnostic foundation for all operations.
