LLM Framework Integrations

Qdrant integrates as a first-class vector store with all major LLM orchestration frameworks:

LangChain

Partner package langchain-qdrant. Dense, sparse, and hybrid retrieval. Most popular integration by usage.

LlamaIndex

QdrantVectorStore for document ingestion, indexing, and query pipelines.

Haystack / AutoGen / CrewAI

Integrations for NLP pipelines and multi-agent systems. Qdrant as persistent memory.

💡
Most common pattern: RAG pipeline with LangChain + Qdrant + OpenAI. Documents are chunked, embedded, stored in Qdrant with metadata payloads, and retrieved at query time to augment LLM prompts. Works with any embedding provider and any LLM.

Official Tools

Qdrant Cloud

Fully managed deployment. AWS, GCP, Azure. Free tier. Terraform-enabled API.

Cloud Inference

Unified embedding + storage. Send raw text/images, get vectors + search in one API call.

Qdrant Edge

Retrieval on edge devices. Offline-capable, privacy-preserving, no server dependency.

FastEmbed

Lightweight ONNX embedding library. Local inference, no GPU needed. Text, image, sparse.
