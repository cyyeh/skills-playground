Qdrant (pronounced "quadrant") is an open-source vector similarity search engine and vector database built entirely in Rust. It stores, indexes, and searches high-dimensional vector embeddings alongside arbitrary JSON metadata (called payloads), enabling applications to find semantically similar items at scale. Think of it as a specialized database that understands "meaning" rather than just exact matches -- you give it a vector representing a concept, and it returns the most similar vectors in its collection.

Founded by Qdrant Solutions GmbH in Berlin in 2021, Qdrant was born from the recognition that AI applications -- particularly those using Large Language Models -- need a purpose-built storage layer that combines fast approximate nearest neighbor search with rich metadata filtering. The result is a database engine that can search through billions of vectors with sub-millisecond latency while simultaneously filtering by structured metadata, all with the memory safety and performance guarantees of Rust.

### What It Is

Qdrant is a vector database that stores numerical embeddings (dense vectors, sparse vectors, and multi-vectors) generated by machine learning models and provides blazing-fast similarity search over them. You send it vectors with metadata via a REST or gRPC API, and it indexes them using a custom implementation of the HNSW (Hierarchical Navigable Small World) algorithm. When you query it, Qdrant traverses this graph to find the nearest neighbors to your query vector -- optionally filtering by metadata conditions -- and returns results ranked by similarity score.

### Who It's For

Qdrant is built for AI/ML engineers building retrieval-augmented generation (RAG) systems, recommendation engines, semantic search, and any application that needs to find "similar things" in a large corpus. It's ideal for teams that need production-grade vector search with complex filtering, multi-tenancy, horizontal scaling, and the operational reliability of a Rust-based system. It's also used by researchers and startups prototyping AI applications, thanks to its generous free tier on Qdrant Cloud and straightforward Docker deployment.

### The One-Sentence Pitch

Qdrant is a Rust-built vector database that delivers high-performance similarity search with advanced filtering, quantization, and distributed scaling -- purpose-built as the retrieval layer for modern AI applications.
