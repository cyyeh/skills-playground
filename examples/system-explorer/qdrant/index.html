Quick Start

Get Qdrant running in seconds with Docker, then create your first collection and run a similarity search.

# Pull and run Qdrant with persistent storage
docker pull qdrant/qdrant
docker run -p 6333:6333 -p 6334:6334 \
    -v "$(pwd)/qdrant_storage:/qdrant/storage:z" \
    qdrant/qdrant

# REST API at localhost:6333
# Web Dashboard at localhost:6333/dashboard
# gRPC API at localhost:6334

Then install the Python client and start exploring: pip install qdrant-client
