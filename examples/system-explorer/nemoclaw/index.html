Quick Start

Get a sandboxed AI agent running with a single command. The installer handles Docker, Node.js, and the interactive onboarding wizard.

curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash

After onboarding, connect to your agent and start chatting:

nemoclaw my-assistant connect
openclaw tui

Explore the Core Concepts page to understand what happens under the hood, or jump to Implementation Details for the full configuration guide.
