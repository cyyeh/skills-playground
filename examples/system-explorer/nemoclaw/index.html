NemoClaw

Quick Start

Install NemoClaw and launch a sandboxed OpenClaw agent with a single command.

bash 
# Install NemoClaw (handles all dependencies)
curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash

# Connect to your sandboxed agent
nemoclaw my-assistant connect

# Or use the interactive chat TUI
openclaw tui
