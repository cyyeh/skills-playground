Quick Start

NemoClaw wraps OpenClaw with enterprise-grade security. Install and onboard in two commands:

# Install NemoClaw
curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash

# Run the interactive onboarding wizard
nemoclaw onboard

# Connect to your sandbox
nemoclaw my-sandbox connect

# Start your secured agent
openclaw

Requires Linux (Ubuntu 22.04+), Docker 20.x+, Node.js 22.16+, and at least 8 GB RAM.
