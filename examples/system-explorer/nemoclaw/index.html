Quick Start

NemoClaw installs with a single command. The interactive wizard handles inference provider setup, credential configuration, and sandbox image download.

bash 
curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash

# Launch a sandbox
nemoclaw launch

# Connect to the agent TUI
openclaw tui

# Check sandbox status
nemoclaw status

Minimum requirements: Ubuntu 22.04+, 4 vCPU, 8 GB RAM, 20 GB disk, Docker. No GPU required for cloud inference routing.
