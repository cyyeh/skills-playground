Quick Start

Get a sandboxed OpenClaw instance running on a Linux machine in under five minutes. All you need is Docker and a single command.

Bash 
# Install NemoClaw (checks prerequisites, runs onboarding wizard)
curl -fsSL https://www.nvidia.com/nemoclaw.sh | bash

# Connect to your sandboxed agent
nemoclaw my-assistant connect

# Start the OpenClaw TUI inside the sandbox
openclaw tui

The onboarding wizard will guide you through selecting an inference provider, configuring credentials (stored on the host, never in the sandbox), and applying security policies. Minimum requirements: 4 vCPU, 8 GB RAM, 20 GB disk.
