Quick Start

Get up and running with DuckDB in seconds. No server, no configuration -- just install and start querying.

Python 
import duckdb

# Query a Parquet file directly -- no loading step needed
result = duckdb.sql("""
    SELECT region, SUM(amount) as total
    FROM 'sales.parquet'
    GROUP BY region
    ORDER BY total DESC
""")

result.show()
💡
Tip: DuckDB can also query CSV, JSON, and remote files on S3 or HTTP endpoints with the same syntax. Just change the file path.
