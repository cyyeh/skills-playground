Quick Start

Get started with DuckDB in seconds. No server, no configuration -- just install and query.

Python 
# Install DuckDB
# pip install duckdb

import duckdb

# Query a CSV file directly -- no import needed
result = duckdb.sql("""
    SELECT region, SUM(amount) as total
    FROM 'sales_data.csv'
    GROUP BY region
    ORDER BY total DESC
""")

result.show()
