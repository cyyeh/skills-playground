Quick Start

Up and Running in Seconds

No server to install, no configuration to manage. Just pip install duckdb and go.

Python 
import duckdb

# Query a Parquet file directly — no server, no setup
result = duckdb.sql("SELECT region, SUM(amount) FROM 'sales.parquet' GROUP BY region")
print(result.fetchdf())  # Returns a pandas DataFrame
