Ecosystem & Integrations

Official Extensions

DuckDB ships with ~24 core extensions. Install with INSTALL name; LOAD name;.

ExtensionPurpose
parquetRead/write Parquet with predicate pushdown and column pruning
jsonQuery JSON files directly, JSON extraction functions
httpfsRead files from HTTP/S3/GCS/Azure
spatialGIS operations: ST_Point, ST_Distance, R-tree indexes
icebergApache Iceberg table support
postgres_scannerQuery live PostgreSQL databases directly
excelRead .xlsx files as tables
ftsFull-text search indexes
vssHNSW indexes for vector similarity search

Key Ecosystem Tools

☁

MotherDuck

Cloud-native DuckDB with hybrid execution, shared databases, and multi-user access. Connect with md: prefix.

🏗

dbt-duckdb

Official dbt adapter. Run dbt models locally or against MotherDuck. CI/CD without a cloud warehouse.

📊

Evidence.dev

Code-first BI tool using DuckDB as local query engine. Write SQL in Markdown, get dashboards.

🐘

pg_duckdb

Embeds DuckDB inside PostgreSQL. Analytical queries auto-route to DuckDB engine for dramatic speedups.

🐍

Ibis

Python DataFrame API that compiles to DuckDB SQL locally or BigQuery/Snowflake remotely.

Language Client Libraries

DuckDB provides official bindings for 11+ languages. The Python client is most popular, with zero-copy integration with Pandas, Polars, and Arrow.

Python
import duckdb
# Connect to MotherDuck cloud
con = duckdb.connect("md:my_database?motherduck_token=<token>")
con.sql("SELECT * FROM cloud_table LIMIT 10").show()
